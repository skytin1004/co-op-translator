# Augmentation の概要

Azure Machine Learning の [augmentation library](https://learn.microsoft.com/azure/machine-learning/how-to-use-augmentation-library) は、強力で拡張性の高いツールセットを提供し、データセットを変換または拡張して AI モデルのパフォーマンスを向上させます。これらの変換はトレーニング時にデータにランダム性を導入し、モデルの過適合を減らし、精度を高めるのに役立ちます。

- 画像変換: 回転、フリップ、切り抜き、色の補正、拡大縮小などの様々な画像変換を行います。
- テキスト変換: シノニム置換、ランダム挿入、テキストのマスキングなどを含め、自然言語データを拡張します。
- 組み合わせ: 複数の変換をパイプラインで連携させ、複雑な augmentation 戦略を作成します。

## 使用例

- 画像分類モデルの学習でランダムな翻転や切り抜きを使用してデータの多様性を増やす
- テキスト分類評価時に文のノイズを増やしてモデルのロバスト性を評価
- トレーニング時間中にデータセット全体を変換する代わりにバッチ単位で動的に変換を適用

詳細およびサポートされている変換については、[augmentation library ドキュメント](https://learn.microsoft.com/azure/machine-learning/how-to-use-augmentation-library)を参照してください。
