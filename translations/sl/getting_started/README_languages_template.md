# 🌐 Večjezična podpora (predloga)

Vzdrževalci: Spodnji blok je »primer za vse jezike«, ki ga upravlja Co-op Translator.

- Če želite, da Co-op Translator samodejno posodablja ta seznam pri vsakem zagonu ukaza `translate` (za kateri koli jezik), pustite obe komentarni oznaki nespremenjeni.
- Če želite prikazati le podnabor jezikov, izbrišite obe komentarni oznaki in odstranite jezike, ki jih nočete prikazati. Po odstranitvi oznak Co-op Translator ne bo več samodejno zamenjeval tega dela.

- Sedaj odsek vsebuje priporočilo "Raje klonirati lokalno?", ki uporabnikom pomaga klonirati brez velike obremenitve z velikimi prevodi. Priporočilo lahko prilagodite s URL-jem vašega repozitorija, tako da zaženete, na primer:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
