# 🌐 Dukungan Multi-Bahasa (Template)

Pemelihara: Blok di bawah ini adalah contoh "semua bahasa" yang dikelola oleh Co-op Translator.

- Jika Anda ingin Co-op Translator menjaga daftar ini tetap terbaru secara otomatis saat Anda menjalankan `translate` (pilihan bahasa apa pun), biarkan kedua penanda komentar seperti semula.
- Jika Anda hanya ingin menampilkan sebagian bahasa, hapus kedua penanda komentar tersebut dan hilangkan bahasa yang tidak ingin Anda tampilkan. Setelah menghapus penanda tersebut, Co-op Translator tidak akan lagi mengganti bagian ini secara otomatis.

- Bagian ini sekarang mencakup saran "Lebih Suka Clone Secara Lokal?" untuk membantu pengguna meng-clone tanpa beban terjemahan yang besar. Anda dapat menyesuaikan saran ini dengan URL repositori Anda dengan menjalankan, misalnya:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
