# 🌐 Dukungan Multi-Bahasa (Template)

Pemelihara: Blok di bawah ini adalah contoh "semua bahasa" yang dikelola oleh Co-op Translator.

- Jika Anda ingin Co-op Translator menjaga daftar ini tetap terbaru secara otomatis saat Anda menjalankan `translate` (pemilihan bahasa apa pun), biarkan dua penanda komentar seperti semula.
- Jika Anda hanya ingin menampilkan sebagian bahasa, hapus dua penanda komentar tersebut dan hapus bahasa yang tidak ingin Anda cantumkan. Setelah menghapus penanda, Co-op Translator tidak akan lagi mengganti bagian ini secara otomatis.

- Bagian ini sekarang menyertakan saran "Lebih Suka Clone Secara Lokal?" untuk membantu pengguna meng-clone tanpa payload terjemahan besar. Anda dapat mempersonalisasi saran ini dengan URL repositori Anda dengan menjalankan misalnya:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
