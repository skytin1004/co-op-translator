# Bahasa yang didukung

Tabel di bawah ini mencantumkan bahasa yang saat ini didukung oleh **Co-op Translator**. Ini mencakup kode bahasa, nama bahasa, dan masalah yang diketahui terkait dengan setiap bahasa. Jika Anda ingin menambahkan dukungan untuk bahasa baru, silakan tambahkan kode bahasa, nama, dan font yang sesuai dalam file `font_language_mappings.yml` yang terletak di `src/co_op_translator/fonts/` dan kirimkan pull request setelah pengujian.
