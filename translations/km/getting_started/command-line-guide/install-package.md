# ដំឡើងកញ្ចប់អ្នកបកប្រែ Co-op

**Co-op Translator** គឺជាឧបករណ៍បញ្ជា​បន្ទាត់ (CLI) ដែលរចនាឡើងដើម្បីជួយអ្នកបកប្រែឯកសារ markdown និងរូបភាពទាំងអស់ក្នុងគម្រោងរបស់អ្នកទៅជាភាសាច្រើន។ មេស៊ីងនេះនឹងណែនាំអ្នកដល់ការកំណត់រចនាសម្ព័ន្ធនិងរត់វានៅសម្រាប់ករណីប្រើប្រាស់ផ្សេងៗ។

### បង្កើតបរិស្ថាន الافتراضي

អ្នកអាចបង្កើតបរិស្ថាន الافتراضيដោយប្រើ `pip` ឬ `Poetry`។ សូមវាយពាក្យបញ្ជាមួយក្នុងចំណោមខាងក្រោមនៅក្នុងផ្ទាំងបញ្ជារបស់អ្នក។

#### ប្រើ pip

```bash
python -m venv .venv
```

#### ប្រើ Poetry

```bash
poetry init
```

### បើកបរិស្ថាន الافتراضي

បន្ទាប់ពីបង្កើតបរិស្ថាន الافتراضي សូមបើកវា។ ជំហាននីមួយៗមានការប្រែប្រួលអាស្រ័យលើប្រព័ន្ធប្រតិបត្តិការរបស់អ្នក។ សូមវាយពាក្យបញ្ជាខាងក្រោមនៅក្នុងផ្ទាំងបញ្ជារ។

#### សម្រាប់ pip និង Poetry ទាំងពីរ

- Windows:

    ```bash
    .venv\Scripts\activate
    ```

- Mac/Linux:

    ```bash
    source .venv/bin/activate
    ```

#### ប្រើ Poetry

1. ប្រសិនបើអ្នកបានបង្កើតបរិស្ថានដោយ Poetry សូមវាយពាក្យបញ្ជាខាងក្រោមនៅក្នុងផ្ទាំងបញ្ជារដើម្បីបើកវា។

    ```bash
    poetry shell
    ```

### ដំឡើងកញ្ចប់ និងកញ្ចប់ដែលត្រូវការ

ពេលបរិស្ថាន الافتراضي របស់អ្នកត្រូវបានកំណត់រួច និងបើកសកម្ម អ្នកត្រូវដំឡើងការពឹងផ្អែកចាំបាច់។

### ដំឡើងយ៉ាងរហ័ស

ដំឡើងតាមរយៈ Co-Op Translator តាមរបៀប pip

```
pip install co-op-translator
```
 ឬ

ដំឡើងតាមរយៈ poetry
```
poetry add co-op-translator
```

#### ប្រើ pip (ពី requirements.txt) ប្រសិនបើអ្នកបានក្លូន repo នេះ

> [!NOTE]
> សូមកុំធ្វើការនេះប្រសិនបើអ្នកបានដំឡើង co-op translator តាមរយៈការដំឡើងយ៉ាងរហ័ស។

1. ប្រសិនបើអ្នកប្រើ pip សូមវាយពាក្យបញ្ជាខាងក្រោមនៅក្នុងផ្ទាំងបញ្ជាររបស់អ្នក។ វានឹងដំឡើងកញ្ចប់ដែលត្រូវការដែលបានបញ្ជាក់ក្នុងឯកសារ `requirements.txt` ដោយស្វ័យប្រវត្តិ៖

    ```bash
    pip install -r requirements.txt
    ```

#### ប្រើ Poetry (ពី pyproject.toml)

1. ប្រសិនបើអ្នកប្រើ Poetry សូមវាយពាក្យបញ្ជាខាងក្រោមនៅក្នុងផ្ទាំងបញ្ជាររបស់អ្នក។ វានឹងដំឡើងកញ្ចប់ដែលត្រូវការដែលបានបញ្ជាក់ក្នុងឯកសារ `pyproject.toml` ដោយស្វ័យប្រវត្តិ៖

    ```bash
    poetry install
    ```
