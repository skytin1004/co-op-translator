# តម្លើងកញ្ចប់ Co-op translator

**Co-op Translator** គឺជាឧបករណ៍ command-line interface (CLI) ដែលរចនាឡើងដើម្បីជួយអ្នកបកប្រែកំណត់ត្រាម៉ាកដោន និងរូបភាពទាំងអស់ក្នុងគម្រោងរបស់អ្នកទៅជาทាសាច្រើន។ មេរៀននេះនឹងណែនាំអ្នកពីការកំណត់ translator និងការប្រើវាសម្រាប់បំណងប្រើប្រាស់ផ្សេងៗ។

### បង្កើតបរិបទ virtual

អ្នកអាចបង្កើតបរិបទ virtual ដោយប្រើ `pip` ឬ `Poetry`។ សូមវាយពាក្យបញ្ជាមួយក្នុងចំណោមបញ្ជាខាងក្រោមនៅក្នុង terminal របស់អ្នក។

#### ប្រើ pip

```bash
python -m venv .venv
```

#### ប្រើ Poetry

```bash
poetry init
```

### បើកបរិបទ virtual

បន្ទាប់ពីបង្កើតបរិបទ virtual អ្នកត្រូវតែបើកវា។ ជំហាននេះខុសគ្នាតាមប្រព័ន្ធប្រតិបត្តិការរបស់អ្នក។ សូមវាយពាក្យបញ្ជាខាងក្រោមនៅក្នុង terminal របស់អ្នក។

#### សម្រាប់ ทั้ง pip และ Poetry

- Windows:

    ```bash
    .venv\Scripts\activate
    ```

- Mac/Linux:

    ```bash
    source .venv/bin/activate
    ```

#### ប្រើ Poetry

1. ប្រសិនបើអ្នកបានបង្កើតបរិបទប្រើ Poetry សូមវាយពាក្យបញ្ជាខាងក្រោមក្នុង terminal ដើម្បីបើកវា។

    ```bash
    poetry shell
    ```

### ការតម្លើង Package និង Package ចាំបាច់

បន្ទាប់​ពី​បរិបទ virtual របស់​អ្នក​ត្រូវ​បាន​កត់ត្រា​ថ្លៃ និង​បាន​បើក​ ជំហាន​បន្ទាប់​គឺ​តម្លើងការពឹងផ្អែកដែលចាំបាច់។

### តម្លើង​យ៉ាង​រហ័ស

តម្លើងតាមរយៈ Co-Op Translator តាម pip

```
pip install co-op-translator
```
 ឬ

តម្លើងតាមរយៈ poetry
```
poetry add co-op-translator
```

#### ប្រើ pip (ពី requirements.txt) ប្រសិនបើអ្នក clone repo នេះ

> [!NOTE]
> សូមកុំធ្វើនូវបែបនេះ ប្រសិនបើអ្នកបានតម្លើង co-op translator តាមរយៈការតម្លើងយ៉ាងរហ័ស។

1. ប្រសិនបើអ្នកប្រើ pip សូមវាយពាក្យបញ្ជាខាងក្រោមក្នុង terminal។ វានឹងតម្លើងជាស្វ័យប្រវត្ត នូវ package ដែលត្រូវការដែលបានបញ្ជាក់នៅក្នុងឯកសារ `requirements.txt`៖

    ```bash
    pip install -r requirements.txt
    ```

#### ប្រើ Poetry (ពី pyproject.toml)

1. ប្រសិនបើអ្នកប្រើ Poetry សូមវាយពាក្យបញ្ជាខាងក្រោមក្នុង terminal។ វានឹងតម្លើងជាស្វ័យប្រវត្ត នូវ package ដែលត្រូវការដែលបានបញ្ជាក់នៅក្នុងឯកសារ `pyproject.toml`៖

    ```bash
    poetry install
    ```
