# របៀបប្រើប្រាស់ចំណុចបញ្ជា Command line interface (CLI) របស់ Co-op Translator

## លក្ខខណ្ឌជាមុន

- **Python 3.10 ឬខ្ពស់ជាងនេះ**: ត្រូវការសម្រាប់ការរត់ Co-op Translator។

## តារាងមាតិកា

1. [បង្កើតឯកសារ '.env' នៅថតមេ](./create-env-file.md)
   - រួមបញ្ចូលកូនសោដែលចាំបាច់សម្រាប់សេវាដែលបានជ្រើសរើសរបស់ម៉ូដែលភាសា។
   - ប្រសិនបើមិនបញ្ចូលកូនសោ Azure Computer Vision ឬបានបញ្ជាក់ `-md` ជា, កម្មវិធីបកប្រែ​នឹងដំណើរការដោយរបៀបទៅលើ Markdown តែប៉ុណ្ណោះ។
1. [ដំឡើងកញ្ចប់ Co-op translator](./install-package.md)
1. [បកប្រែមុខងាររបស់អ្នកប្រើ Co-op Translator](./translator-your-project.md)
