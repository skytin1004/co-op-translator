# របៀបប្រើប្រាស់ចំណុចបញ្ជា command line interface (CLI) របស់ Co-op Translator

## លក្ខខណ្ឌមុនខ្លះ

- **Python 3.10 ឬខ្ពស់ជាងនេះ**: ត្រូវការសម្រាប់រត់ Co-op Translator។

## តារាងមាតិកា

1. [បង្កើត​ឯកសារ '.env' នៅក្នុងថត root](./create-env-file.md)
   - រួមបញ្ចូល key ដែលចាំបាច់សម្រាប់សេវាម៉ូដែលភាសាដែលបានជ្រើស។
   - ប្រសិនបើ key Azure Computer Vision ត្រូវបានបោះបង់ ឬបានបញ្ជាក់ `-md`, កម្មវិធីបកប្រែ នឹងដំណើរការជារបៀប Markdown តែប៉ុណ្ណោះ។
1. [ដំឡើងកញ្ចប់ Co-op translator](./install-package.md)
1. [បកប្រែ​គម្រោង​របស់​អ្នក​ដោយ​ប្រើ Co-op Translator](./translator-your-project.md)
