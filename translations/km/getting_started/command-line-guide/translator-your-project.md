# бһ”бһҖбһ”бҹ’бһҡбҹӮбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Co-op Translator

**Co-op Translator** бһӮбһәбһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһ”бһүбҹ’бһҮбһ¶бһӣбҹҒбһҒ (CLI) бһҠбҹӮбһӣбһҮбһҪбһҷбһўбҹ’бһ“бһҖбһ”бһҖбһ”бҹ’бһҡбҹӮбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё markdown бһ“бһ·бһ„бҖ–бҖӯбҖҜбҖ„бҖәбһҡбһјбһ”бһ—бһ¶бһ–бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһӮбһҳбҹ’бһҡбҹ„бһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бҹ…бһҮбһ¶бһ—бһ¶бһҹбһ¶бһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ” бһ•бҹ’бһ“бҹӮбһҖбһ“бҹҒбҹҮбһҹбҹҶбһҠбҹӮбһ„бһ–бһёбһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһ“бҹҒбҹҮвҖӢ бһҡбҹҖбһ”бһҡбһ¶бһ”бҹӢбһҮбһҳбҹ’бһҡбһҫбһҹ CLI бһ•бҹ’бһҹбҹҒбһ„бҹ— бһ“бһ·бһ„бһ•бҹ’бһҠбһӣбҹӢбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ“бһ¶бһ“бһ¶бҹ”

## бһҖбһҡбһҺбһёбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһ“бһ·бһ„бһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶

бһ“бҹҒбҹҮбһҮбһ¶бһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ‘бһјбһ‘бҹ…бһҒбҹ’бһӣбҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ **Co-op Translator** бһҡбһҪбһҳбһҮбһ¶бһҳбһҪбһҷбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҠбҹӮбһӣбһҹбһҳбһҹбҹ’бһҡбһ”бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбҹ”

### 1. бһ”бһҖбһ”бҹ’бһҡбҹӮбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ (бһ—бһ¶бһҹбһ¶бһҳбһҪбһҷ)

бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҖбһ”бҹ’бһҡбҹӮбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ (бһҜбһҖбһҹбһ¶бһҡ markdown бһ“бһ·бһ„бһҡбһјбһ”бһ—бһ¶бһ–) бһ‘бҹ…бһҮбһ¶бһ—бһ¶бһҹбһ¶бһҳбһҪбһҷ бһҠбҹӮбһӣбһҮбһ¶Korean бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһ”бҹ’бһҡбһҫбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

```bash
translate -l "ko"
```

бһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһ“бҹҒбҹҮбһ“бһ№бһ„бһ”бһҖбһ”бҹ’бһҡбҹӮбһҜбһҖбһҹбһ¶бһҡ markdown бһ“бһ·бһ„бһҡбһјбһ”бһ—бһ¶бһ–бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ‘бҹ…бһҮбһ¶бһ—бһ¶бһҹбһ¶бһҖбһјбһҡбҹүбҹҒ бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһ”бһҖбһ”бҹ’бһҡбҹӮбһҗбҹ’бһҳбһёбҹ—бһҠбҹ„бһҷбһҳбһ·бһ“бһӣбһ»бһ”бһҖбһ¶бһҡбһ”бһҖбһ”бҹ’бһҡбҹӮбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбҹҒбһ…бҹ”

> [!TIP]
>
> бһ…бһ„бҹӢбһҠбһ№бһ„бһҖбһјбһҠбһ—бһ¶бһҹбһ¶бһҠбҹӮбһӣбһҳбһ¶бһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ **Co-op Translator** бһҳбҹӮбһ“бһ‘бҹҒ? бһҹбһјбһҳбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһ•бҹ’бһ“бҹӮбһҖ [Supported Languages](https://github.com/Azure/co-op-translator#supported-languages) бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ repository бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӣбһҳбҹ’бһўбһ·бһҸбһ”бһ“бҹ’бһҗбҹӮбһҳбҹ”

#### бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹ… Phi-3 CookBook

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ **Phi-3 CookBook** бһҒбҹ’бһүбһ»бҹҶбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһңбһ·бһ’бһёбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһ”бһҖбһ”бҹ’бһҡбҹӮбһҖбһјбһҡбҹүбҹҒбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҜбһҖбһҹбһ¶бһҡ markdown бһ“бһ·бһ„бһҡбһјбһ”бһ—бһ¶бһ–бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбҹҒбһ…бҹ”

### 2. бһ”бһҖбһ”бҹ’бһҡбҹӮбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһЎбһҫбһ„бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ—бһ¶бһҹбһ¶

бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҖбһ”бҹ’бһҡбҹӮбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бҹ…бһҮбһ¶бһ—бһ¶бһҹбһ¶бһ…бҹ’бһҡбһҫбһ“ (бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһўбҹҒбһҹбҹ’бһ”бҹүбһ¶бһү бһ”бһ¶бһҡбһ¶бҹҶбһ„ бһ“бһ·бһ„бһўбһ¶бһӣбҹ’бһӣбһәбһҳбҹүбһ„бҹӢ) бһ”бҹ’бһҡбһҫбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһ“бҹҒбҹҮбҹ–


#### бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹ… Phi-3 CookBook

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ **Phi-3 CookBook** бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҷбһҖбһҖбҹҶбһҺбҹӮбһҗбҹ’бһҳбһёбһ”бҹҶбһ•бһ»бһҸ бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһ‘бһ¶бһ“бҹӢбһҹбһҳбҹҗбһҷбһ“бһјбһңбһҖбһ¶бһҡбһ”бҹ’бһҸбһјбһҡбһҗбҹ’бһҳбһёбҹ— бһҒбҹ’бһүбһ»бҹҶбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһңбһ·бһ’бһёбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҖбһ”бҹ’бһҡбҹӮбһҜбһҖбһҹбһ¶бһҡ markdown бһ“бһ·бһ„бһҡбһјбһ”бһ—бһ¶бһ–бһҗбҹ’бһҳбһёбҹ—бҹ”

>    - бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҗбһҸбһҠбҹӮбһӣбһҳбһ¶бһ“бһҜбһҖбһҹбһ¶бһҡбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶ `cd`бҹ”
>    - бһ”бҹ’бһҡбһҫбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҠбһҫбһҳбҹ’бһ”бһёбһӣбһ»бһ”бһҜбһҖбһҹбһ¶бһҡбҹ–
>      ```
>      del /s *filename*
>      ```
>      бһ”бҹ’бһҸбһјбһҡ `filename` бһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҲбҹ’бһҳбҹ„бҹҮбһҜбһҖбһҹбһ¶бһҡбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбҹ” бһҮбһҳбҹ’бһҡбһҫбһҹ `/s` бһ“бһ№бһ„бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҖбҹ’бһ“бһ»бһ„бһҗбһҸбһҡбһ„бһ•бһ„бһҠбҹӮбһҡбҹ”
>
> 2. **бһ”бҹ’бһҡбһҫ PowerShell**:
>    - бһ”бһҫбһҖ PowerShellбҹ”
>    - бһҡбһҸбҹӢбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһ“бҹҒбҹҮбҹ–
>      ```powershell
>      Get-ChildItem -Path "C:\YourPath" -Filter "*filename*" -Recurse | Remove-Item -Force
>      ```
>      бһ”бҹ’бһҸбһјбһҡ `"C:\YourPath"` бһҮбһ¶бһ•бҹ’бһӣбһјбһңбһҗбһҸ бһ“бһ·бһ„ `filename` бһҮбһ¶бһҲбҹ’бһҳбҹ„бҹҮбһҜбһҖбһҹбһ¶бһҡбһ–бһ·бһҹбҹҒбһҹбҹ”
>
> ### бһӣбһҫ macOS/Linux:
> 1. **бһ”бҹ’бһҡбһҫ Terminal**:
>   - бһ”бһҫбһҖ Terminalбҹ”
>   - бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҗбһҸбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ `cd`бҹ”
>   - бһ”бҹ’бһҡбһҫбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶ `find` бҹ–
>     ```bash
>     find . -type f -name "*filename*" -delete
>     ```
>     бһ”бҹ’бһҸбһјбһҡ `filename` бһҮбһ¶бһҲбҹ’бһҳбҹ„бҹҮбһҜбһҖбһҹбһ¶бһҡбһ–бһ·бһҹбҹҒбһҹбҹ”
>
> бһҮбһ¶бһңбһ·бһ’бһ¶бһ“бһҖбһ¶бһҡбһ”бһҳбҹ’бһҡбһ»бһ„бһ‘бһ»бһҖбһҳбһ»бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҜбһҖбһҹбһ¶бһҡбһҳбһ»бһ“бһӣбһ»бһ” бһҠбһҫбһҳбҹ’бһ”бһёбһҮбҹҖбһҹбһңбһ¶бһ„бһҖбһ¶бһҡбһ”бһ¶бһҸбҹӢбһ”бһ„бҹӢбһҜбһҖбһҹбһ¶бһҡбһҠбҹ„бһҷбһҖбҹҶбһ бһ»бһҹбҹ”
>
> бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһӣбһ»бһ”бһҜбһҖбһҹбһ¶бһҡбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҸбһјбһҡ бһҡбһҸбҹӢ `translate -l` бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҳбҹ’бһҠбһ„бһ‘бҹҖбһҸ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҜбһҖбһҹбһ¶бһҡбһҗбҹ’бһҳбһёбҹ—бҹ”
