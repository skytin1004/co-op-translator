# 🌐 ការ​គាំទ្រភាសាច្រើន (កម្រងតម្រង់)

អ្នកថែទាំ៖ ប្លុកខាងក្រោមគឺជាគំរូ "ភាសាទាំងអស់" ដែលគ្រប់គ្រងដោយ Co-op Translator។

- ប្រសិនបើអ្នកចង់ឲ្យ Co-op Translator រក្សាបញ្ជីនេះឲ្យទាន់សម័យដោយស្វ័យប្រវត្តិបន្ទាប់ពីអ្នករត់ `translate` (ជ្រើសរើសភាសាណាមួយ), សូមរក្សាទុកសញ្ញាកំណត់មតិទាំងពីរដូចតាមដើម។
- ប្រសិនបើអ្នកចង់បង្ហាញតែភាសាខ្លះៗ, សូមលុបសញ្ញាកំណត់មតិទាំងពីរនិងលុបភាសាណាមួយដែលអ្នកមិនចង់បញ្ចូលក្នុងបញ្ជី។ បន្ទាប់ពីលុបសញ្ញាកំណត់មតិ, Co-op Translator នឹងមិនប្តូរចំណុចនេះដោយស្វ័យប្រវត្តិទៀតទេ។

- បញ្ចាំងឥឡូវនេះមានដំណឹង "ចូលចិត្តចម្លងនៅក្នុងកុំព្យូទ័រផ្ទាល់?" ដើម្បីជួយអ្នកប្រើប្រាស់ចម្លងដោយគ្មានទម្ងន់ការប្រែក្លាយជាភាសាធំនោះទេ។ អ្នកអាចប្តូរការណែនាំនេះបានដោយបញ្ចូល URL អ្នករក្សាទុករបស់អ្នក ដោយរត់ ឧទាហរណ៍ៈ
