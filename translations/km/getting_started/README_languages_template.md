# 🌐 សហគមន៍ភាសាច្រើន (គំរូ)

អ្នកថែរក្សា៖ ប្លុកខាងក្រោមគឺជាឧទាហរណ៍ "ភាសាទាំងអស់" ដែលគ្រប់គ្រងដោយ Co-op Translator។

- ប្រសិនបើអ្នកចង់ឲ្យ Co-op Translator ជួយថែរក្សាបញ្ជីនេះឲ្យទាន់សម័យទាំងមូលដោយស្វ័យប្រវត្តិពេលអ្នកបញ្ជាទូទៅ `translate` (ជ្រើសរើសភាសាណាមួយ) សូមរក្សាទុកសញ្ញាគំនិតពីរ ជាបែបដើម។
- ប្រសិនបើអ្នកចង់បង្ហាញតែភាសាខ្លះៗប៉ុណ្ណោះ សូមលុបសញ្ញាគំនិតពីរនិងដកភាសាណាមួយដែលអ្នកមិនចង់បញ្ជាក់។ បន្ទាប់ពីលុបសញ្ញាគំនិតនេះ Co-op Translator នឹងមិនបន្តបម្លែងផ្នែកនេះឡើយ។

- ផ្នែកនេះឥឡូវបានរួមបញ្ចូល "ចង់ចម្លងទៅក្នុងម៉ាសុីនក្នុងតំបន់មែនទេ?" ដែលជាអនុសាសន៍ដើម្បីជួយអ្នកប្រើចម្លងដោយគ្មានទម្ងន់សំអាងប្រែសម្រួលធំ។ អ្នកអាចផ្ទាល់ខ្លួនអនុសាសន៍នេះជាមួយ URL ឃ្លាំងរបស់អ្នកដោយបញ្ជាពីរ, ឧទាហរណ៍៖
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
