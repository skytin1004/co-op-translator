# Command reference

The **Co-op Translator** CLI бһ•бҹ’бһҠбһӣбҹӢбһҮбһҳбҹ’бһҡбһҫбһҹбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҸбһјбһҡбһҮбһ¶бһ—бһ¶бһҹбһ¶бһҖбҹ’бһ“бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡвҖӢбһ”бһҖбһ”бҹ’бһҡбҹӮбҹ–

----------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
