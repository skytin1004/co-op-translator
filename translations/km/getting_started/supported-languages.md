# ភាសាដែលគាំទ្រ

តារាងខាងក្រោមបញ្ជាក់ពីភាសាដែល **Co-op Translator** គាំទ្របច្ចុប្បន្ននេះ។ វារួមមានកូដភាសា ឈ្មោះភាសា និងបញ្ហាណាមួយដែលបានគ្រោងទុកសម្រាប់ភាសាទាំងនោះ។ ប្រសិនបើអ្នកចង់បន្ថែមការគាំទ្រសម្រាប់ភាសាថ្មី សូមបន្ថែមកូដភាសា ឈ្មោះ និងអក្សរដែលសមរម្យក្នុងឯកសារ `font_language_mappings.yml` ដែលស្ថិតនៅ `src/co_op_translator/fonts/` ហើយដាក់ទំនាក់ទំនង pull request បន្ទាប់ពីបានធ្វើតេស្តរួច។
