# бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ GitHub Action Co-op Translator (бһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„бһҹбһ¶бһ’бһ¶бһҡбһҺбҹҲ)

**бһўбҹ’бһ“бһҖбһӮбҹ„бһӣбһҠбҹ…бҹ–** бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһ“бҹҒбҹҮбһӮбһәбһҹбҹҶбһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһ¶бһ’бһ¶бһҡбһҺбҹҲ бһ¬бһҜбһҖбһҮбһ“бһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“ бһҠбҹӮбһӣбһҹбһ·бһ‘бҹ’бһ’бһ· GitHub Actions бһӣбҹҶбһ“бһ¶бҹҶбһҠбһҫбһҳбһӮбһәбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢбҹ” бһңбһ¶бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ `GITHUB_TOKEN` бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бҹ”

бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бһҖбһ”бҹ’бһҡбҹӮбһҜбһҖбһҹбһ¶бһҡбһҡбһ”бһҹбҹӢбһғбҹ’бһӣбһ¶бҹҶбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ GitHub Action Co-op Translatorбҹ” бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһ“бҹҒбҹҮбһ“бһ№бһ„бһҺбҹӮбһ“бһ¶бҹҶбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһҮбҹҶбһ бһ¶бһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸ pull requests бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ”бһҖбһ”бҹ’бһҡбҹӮбһ”бһ¶бһ“бһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ– бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһҜбһҖбһҹбһ¶бһҡ Markdown бһ”бҹ’бһҡбһ—бһ– бһ¬бһҡбһјбһ”бһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбҹ”

## бһӣбһҖбҹ’бһҒбһҒбҹҗбһҺбҹ’бһҢбһҳбһ»бһ“бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳ

бһҳбһ»бһ“бһ–бҹҒбһӣбһҖбҹҶбһҺбһҸбҹӢ GitHub Action бһҹбһјбһҳбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҹбһҖбҹ’бһҒбһёбһ”бҹҗбһҸбҹ’бһҡбһҮбҹҶбһ“бһҪбһҷбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ AI бһҠбҹӮбһӣбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҡбһҪбһ…бһҡбһ¶бһӣбҹӢбҹ” 

**1. бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбҹ– бһҹбһҖбҹ’бһҒбһёбһ”бҹҗбһҸбҹ’бһҡбһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶ AI**  
бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҹбһҖбҹ’бһҒбһёбһ”бҹҗбһҸбҹ’бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһ—бһ¶бһҹбһ¶бһҠбҹӮбһӣбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҡбһәбһҳбһҪбһҷбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҷбҹүбһ¶бһ„бһ бҹ„бһ…бһҺбһ¶бһҹбҹӢбҹ–

- **Azure OpenAI**бҹ– бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ“бһӮбһҡбһ”бһ¶бһӣ Endpoint, API Key, бһҲбҹ’бһҳбҹ„бҹҮбһҳбҹүбһјбһҠбҹӮбһӣ/Deployment бһ“бһ·бһ„бһҖбҹҶбһҺбҹӮ APIбҹ”
- **OpenAI**бҹ– бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҖй’Ҙ API, (бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹбҹ– бһўбһ¶бһҷбһҢбһёбһҹбҹ’бһҗбһ¶бһ”бҹҗбһ“, URL бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“, бһҲбҹ’бһҳбҹ„бҹҮбһҳбҹүбһјбһҠбҹӮбһӣ)бҹ”
- бһҹбһјбһҳбһҳбһҫбһӣ [бһҳбҹүбһјбһҠбҹӮбһӣ бһ“бһ·бһ„бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҠбҹӮбһӣбһӮбһ¶бҹҶбһ‘бҹ’бһҡ](../../../../README.md) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бҹҗбһҸбһҳбһ¶бһ“бһӣбһҳбҹ’бһўбһ·бһҸбҹ”

**2. бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹбҹ– бһҹбһҖбҹ’бһҒбһёбһ”бҹҗбһҸбҹ’бһҡбһ’бҹ’бһңбһҫ AI Vision (бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһҖбһ”бҹ’бһҡбҹӮбһҡбһјбһ”бһ—бһ¶бһ–)**

- бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҠбҹӮбһҖбҹӮбһ”бһҖбһ”бҹ’бһҡбҹӮбһўбһҸбҹ’бһҗбһ”бһ‘бһҖбҹ’бһ“бһ»бһ„бһҡбһјбһ”бһ—бһ¶бһ–бһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ”
- **Azure AI Vision**бҹ– бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ“бһӮбһҡбһ”бһ¶бһӣ Endpoint бһ“бһ·бһ„ Subscription Keyбҹ”
- бһ”бҹ’бһҡбһҹбһ·бһ“бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ•бҹ’бһҸбһӣбҹӢбһ‘бҹҒ бһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһ“бҹҒбҹҮбһ“бһ№бһ„бһ”бҹ’бһҡбһҫ [бһҡбһ”бҹҖбһ” Markdown бһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ](../markdown-only-mode.md)бҹ”

## бһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„ бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһҸбҹҶбһҳбҹ’бһӣбһҫбһ„

бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҮбҹҶбһ бһ¶бһ“бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢбһ”бҹ’бһҡбһҫ GitHub Action Co-op Translator бһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҮбһ¶бһҳбһҪбһҷ `GITHUB_TOKEN` бһӣбҹҶбһ“бһ¶бҹҶбһҠбһҫбһҳбҹ”

### бһҮбҹҶбһ бһ¶бһ“ 1бҹ– бһҷбһӣбҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһёбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ (бһ”бҹ’бһҡбһҫ `GITHUB_TOKEN`)

бһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһ“бҹҒбҹҮбһ”бҹ’бһҡбһҫ `GITHUB_TOKEN` бһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҠбҹ„бһҷ GitHub Actionsбҹ” бһҖбһјбһ“бһҹбҹ„бһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһҹбһ·бһ‘бҹ’бһ’бһ·бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҠбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ…бҹҶбһ–бҹ„бҹҮбһғбҹ’бһӣбһ¶бҹҶбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҸбһ¶бһҳбһ”бһҺбҹ’бһҠбһ¶вҖӢбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ бһҠбҹӮбһӣбһҹбҹ’бһҗбһ·бһҸбһҖбҹ’бһ“бһ»бһ„ **бһҮбҹҶбһ бһ¶бһ“ 3**бҹ”

### бһҮбҹҶбһ бһ¶бһ“ 2бҹ– бһҖбҹҶбһҺбһҸбҹӢбһҡбһҝбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ

бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бһүбҹ’бһ…бһјбһӣбһҹбһҖбҹ’бһҒбһёбһ”бҹҗбһҸбҹ’бһҡбһҮбҹҶбһ“бһҪбһҷбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ AI бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҮбһ¶бһҡбһҝбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҠбҹӮбһӣбһ…бһ¶бһҖбҹӢбһҹбҹ„бһҡ бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһғбҹ’бһӣбһ¶бҹҶбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ”

1. бһ…бһјбһӣбһ‘бҹ…бһғбҹ’бһӣбһ¶бҹҶбһ„ GitHub бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ”бҹ’бһҡбһҫбҹ”  
2. бһ‘бҹ…бһҖбһ¶бһ“бҹӢ **Settings** > **Secrets and variables** > **Actions**бҹ”  
3. бһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ **Repository secrets** бһ…бһ»бһ… **New repository secret** бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбһҪбһҷбҹ—бһ“бҹғбһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ AI бһҠбҹӮбһӣбһ‘бһ¶бһҳбһ‘бһ¶бһҡбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ”

### бһҮбҹҶбһ бһ¶бһ“ 3бҹ– бһҖбҹҶбһҺбһҸбҹӢбһҹбһ·бһ‘бҹ’бһ’бһ·бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһҸбҹӢ Workflow

GitHub Action бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҹбһ·бһ‘бҹ’бһ’бһ·бһ’бҹ’бһңбһҫбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҸбһ¶бһҳ `GITHUB_TOKEN` бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ¶бһҖбҹӢбһҹбҹ„бһҡбһҖбһјбһҠ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸ pull requestsбҹ”

1. бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ‘бҹ…бһҖбһ¶бһ“бҹӢ **Settings** > **Actions** > **General**бҹ”  
2. бһҡбһўбһ·бһӣбһ…бһ»бҹҮбһ‘бҹ…бһ•бҹ’бһ“бҹӮбһҖ **Workflow permissions**бҹ”  
3. бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ **Read and write permissions**бҹ” бһңбһ¶бһ•бҹ’бһҸбһӣбҹӢбһҹбһ·бһ‘бһ·бҹ’бһ’ `contents: write` бһ“бһ·бһ„ `pull-requests: write` бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ `GITHUB_TOKEN` бһҖбҹ’бһ“бһ»бһ„бһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһ“бҹҒбҹҮбҹ”  
4. бһ’бһ¶бһ“бһ¶бһҗбһ¶бһ”бһүбҹ’бһ…бһ¶бһҖбҹӢбһ‘бһ·бһүбһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ **Allow GitHub Actions to create and approve pull requests** бһҳбһ¶бһ“бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбҹ”  
5. бһ…бһ»бһ… **Save**бҹ”

![Permission setting](../../../../getting_started/github-actions-guide/imgs/permission-setting.png)

### бһҮбҹҶбһ бһ¶бһ“ 4бҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһҜбһҖбһҹбһ¶бһҡ Workflow

бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ бһ”бһ„бҹ’бһҖбһҫбһҸбһҜбһҖбһҹбһ¶бһҡ YAML бһҠбҹӮбһӣбһҖбҹҶбһҺбһҸбҹӢбһӣбҹҶбһҠбһ¶бһ”бҹӢбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбһҳбҹ’бһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҫ `GITHUB_TOKEN`бҹ”

1. бһ“бҹ…бһҗбһҸ root бһ“бҹғбһғбҹ’бһӣбһ¶бҹҶбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ”бһ„бҹ’бһҖбһҫбһҸбһҗбһҸ `.github/workflows/` бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһңбһ¶бһҳбһ·бһ“бһҳбһ¶бһ“бһ‘бҹҒбҹ”  
2. бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ `.github/workflows/` бһ”бһ„бҹ’бһҖбһҫбһҸбһҜбһҖбһҹбһ¶бһҡбһҲбҹ’бһҳбҹ„бҹҮ `co-op-translator.yml`бҹ”  
3. бһңбһ¶бһҷбһ”бһүбҹ’бһ…бһјбһӣбһҳбһ¶бһҸбһ·бһҖбһ¶бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„ `co-op-translator.yml`бҹ”

```yaml
name: Co-op Translator

on:
  push:
    branches:
      - main

jobs:
  co-op-translator:
    runs-on: ubuntu-latest

    permissions:
      contents: write
      pull-requests: write

    steps:
      - name: Checkout repository
        uses: actions/checkout@v4
        with:
          fetch-depth: 0

      - name: Set up Python
        uses: actions/setup-python@v4
        with:
          python-version: '3.10'

      - name: Install Co-op Translator
        run: |
          python -m pip install --upgrade pip
          pip install co-op-translator

      - name: Run Co-op Translator
        env:
          PYTHONIOENCODING: utf-8
          # === AI Service Credentials ===
          AZURE_AI_SERVICE_API_KEY: ${{ secrets.AZURE_AI_SERVICE_API_KEY }}
          AZURE_AI_SERVICE_ENDPOINT: ${{ secrets.AZURE_AI_SERVICE_ENDPOINT }}
          AZURE_OPENAI_API_KEY: ${{ secrets.AZURE_OPENAI_API_KEY }}
          AZURE_OPENAI_ENDPOINT: ${{ secrets.AZURE_OPENAI_ENDPOINT }}
          AZURE_OPENAI_MODEL_NAME: ${{ secrets.AZURE_OPENAI_MODEL_NAME }}
          AZURE_OPENAI_CHAT_DEPLOYMENT_NAME: ${{ secrets.AZURE_OPENAI_CHAT_DEPLOYMENT_NAME }}
          AZURE_OPENAI_API_VERSION: ${{ secrets.AZURE_OPENAI_API_VERSION }}
          OPENAI_API_KEY: ${{ secrets.OPENAI_API_KEY }}
          OPENAI_ORG_ID: ${{ secrets.OPENAI_ORG_ID }}
          OPENAI_CHAT_MODEL_ID: ${{ secrets.OPENAI_CHAT_MODEL_ID }}
          OPENAI_BASE_URL: ${{ secrets.OPENAI_BASE_URL }}
        run: |
          # =====================================================================
          # IMPORTANT: Set your target languages here (REQUIRED CONFIGURATION)
          # =====================================================================
          # Example: Translate to Spanish, French, German. Add -y to auto-confirm.
          translate -l "es fr de" -y  # <--- MODIFY THIS LINE with your desired languages

      - name: Create Pull Request with translations
        uses: peter-evans/create-pull-request@v5
        with:
          token: ${{ secrets.GITHUB_TOKEN }}
          commit-message: "рҹҢҗ Update translations via Co-op Translator"
          title: "рҹҢҗ Update translations via Co-op Translator"
          body: |
            This PR updates translations for recent changes to the main branch.

            ### рҹ“Ӣ Changes included
            - Translated contents are available in the `translations/` directory
            - Translated images are available in the `translated_images/` directory

            ---
            рҹҢҗ Automatically generated by the [Co-op Translator](https://github.com/Azure/co-op-translator) GitHub Action.
          branch: update-translations
          base: main
          labels: translation, automated-pr
          delete-branch: true
          add-paths: |
            translations/
            translated_images/
```
4.  **бһ”бҹ’бһҠбһјбһҡбһҸбһ¶бһҳбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡ Workflowбҹ–**  
  - **[!IMPORTANT] бһ—бһ¶бһҹбһ¶бһӮбҹ„бһӣбһҠбҹ…бҹ–** бһ“бҹ…бһҮбҹҶбһ бһ¶бһ“ `Run Co-op Translator` бһўбҹ’бһ“бһҖ **бһҸбҹ’бһҡбһјбһңбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ“бһ·бһ„бһҖбҹӮбһ”бҹ’бһҡбҹӮбһ”бһүбҹ’бһҮбһёбһҖбһјбһҠбһ—бһ¶бһҹбһ¶** бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһүбҹ’бһҮбһ¶ `translate -l "..." -y` бһІбҹ’бһҷбһҹбһҳбһҡбһҳбҹ’бһҷбһҮбһ¶бһҳбһҪбһҷбһӮбһҳбҹ’бһҡбҹ„бһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһ”бһүбҹ’бһҮбһёбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ (`ar de es...`) бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҮбҹҶбһ“бһҪбһҹ бһ¬бһҖбҹӮбһ”бҹ’бһҡбҹӮбҹ”  
  - **бһ§бһ”бһҖбһҡбһҺбҹҚбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳ (`on:`):** бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ”бһҫбһҖбһҖбһ¶бһҡбһ„бһ¶бһҡбһ“бҹҒбҹҮбһ“бҹ…бһ–бҹҒбһӣбһҳбһ¶бһ“ push бһҗбҹ’бһҳбһёбһ‘бҹ…бһҖбһ¶бһ“бҹӢ `main`бҹ” бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһғбҹ’бһӣбһ¶бҹҶбһ„бһ’бҹҶбһ“бҹҒбҹҮ бһўбһ¶бһ…бһ”бһ“бҹ’бһҗбҹӮбһҳ `paths:` бһҮбһ¶ filter (бһҳбһҫбһӣбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҳбһ·бһ“бһ”бһ¶бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһӣбһ¶бһҷбһЎбҹҶбһҖбҹ’бһ“бһ»бһ„ YAML) бһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһ“бҹҒбҹҮбһҸбҹӮбһӣбһҫбһ–бҹҒбһӣбһҠбҹӮбһӣбһҜбһҖбһҹбһ¶бһҡбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’ (бһ§. бһҜбһҖбһҹбһ¶бһҡбһҮбҹҶбһ“бһҪбһҷбһ”бҹ’бһҡбһ—бһ–) бһҳбһ¶бһ“бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡ бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ¶бһҡбһ–бһ¶бһҡбһ–бҹҒбһӣбһҡбһ”бһҹбҹӢ runnerбҹ”  
  - **бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӣбҹҶбһўбһ·бһҸ PRбҹ–** бһўбһ¶бһ…бһҖбҹӮ `commit-message`, `title`, `body`, бһҲбҹ’бһҳбҹ„бҹҮ`branch` бһ“бһ·бһ„ `labels` бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҮбҹҶбһ бһ¶бһ“ `Create Pull Request` бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбҹ”

> **бһҖбҹҶбһҺбһҸбҹӢбһ–бҹҒбһӣбһҡбһҸбҹӢ GitHub-hosted Runner:**  
> GitHub-hosted runners бһҠбһјбһ…бһҮбһ¶ `ubuntu-latest` бһҳбһ¶бһ“ **бһҖбҹҶбһҺбһҸбҹӢбһ–бҹҒбһӣбһўбһҸбһ·бһ”бһҡбһҳбһ¶бһҡбһҸбҹ’бһҡбһ№бһҳ 6 бһҳбҹүбҹ„бһ„**бҹ”  
> бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһғбҹ’бһӣбһ¶бҹҶбһ„бһҜбһҖбһҹбһ¶бһҡбһ’бҹҶ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһҖбһ”бҹ’бһҡбҹӮбһ бһҪбһҹ 6 бһҳбҹүбҹ„бһ„ бһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһүбҹ’бһ…бһ”бҹӢбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бҹ”  
> бһҠбһҫбһҳбҹ’бһ”бһёбһҮбҹҖбһҹбһңбһ¶бһ„бһ“бҹҒбҹҮ бһҹбһјбһҳбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бҹ–  
> - бһ”бҹ’бһҡбһҫ **self-hosted runner** (бһӮбҹ’бһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһ–бҹҒбһӣ)  
> - бһ”бһ“бҹ’бһҗбһҷбһ…бҹҶбһ“бһҪбһ“бһ—бһ¶бһҹбһ¶бһӮбҹ„бһӣбһҠбҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһҠбҹҶбһҺбһҫбһҡ  

бһ–бҹҒбһӣбһҠбҹӮбһӣбһҜбһҖбһҹбһ¶бһҡ `co-op-translator.yml` бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„бһҹбһ¶бһҒбһ¶ main бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ (бһ¬бһҹбһ¶бһҒбһ¶бһҠбҹӮбһӣбһҖбҹҶбһҺбһҸбҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳ `on:`) бһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһ“бҹҒбҹҮбһ“бһ№бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ“бҹ…бһ–бҹҒбһӣбһҳбһ¶бһ“бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҹбһ¶бһҒбһ¶бһ“бҹ„бҹҮ (бһ бһҫбһҷбһҸбҹ’бһҡбһјбһңбһ“бһ№бһ„ filter `paths` бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢ)бҹ”
