ច្បាប់ (ត្រូវប្តេជ្ញារត់តាម):
1) រក្សា Markdown links ដដែល: [text](../../../../../../src/co_op_translator/templates/language/URL) -> [translated text](../../../../../../src/co_op_translator/templates/language/same URL)។
2) មិនដែលសរសេរឡើងវិញ links ជាអត្ថបទសាមញ្ញទេ (ឧ. 「text」（URL）, text (URL))។
3) ប្រែតែ link text ផ្ទាល់; រក្សា Markdown វិធីសាស្រ្ត និង URL ដដែល។
4) មិនបន្ថែម 「」ជុំវិញ Markdown link លុះត្រាតែវាក្នុងវេយ្យាករណ៍ក្រៅ-link សម្រាប់បំពេញត្រូវការ ។
