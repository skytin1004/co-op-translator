## бһҡбһ”бҹҖбһ”бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ”бһҖбһ”бҹ’бһҡбҹӮ

Co-op Translator бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҳбһ¶бһҸбһ·бһҖбһ¶бһ”бһҖбһ”бҹ’бһҡбҹӮбһҮбһ¶бһ‘бҹ’бһҡбһ–бҹ’бһҷбһҹбһҳбҹ’бһ”бһҸбҹ’бһҸбһ·бһҹбһјбһ бҹ’бһңбһңбҹӮбһҡбһҖбһ¶бһҡбһ”бҹ’бһҡбһ—бҹҒбһ‘ **бһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҮбһ¶бһҖбҹҶбһҺбҹӮ** бһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶вҖӢбһҜбһҖбһҹбһ¶бһҡбһҹбҹ’бһҗбһ·бһҸбһҹбҹ’бһҗбҹҒбһҡбһЎбһҫбһҷбҹ”

бһ§бһ”бһҖбһҡбһҺбҹҚбһ“бҹҒбҹҮбһҸбһ¶бһҳбһҠбһ¶бһ“бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҡбһ”бһҹбҹӢ Markdown, бһҡбһјбһ”бһ—бһ¶бһ– бһ“бһ·бһ„ бһҹбҹҖбһңбһ—бҹ…бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһ”бһҖбһ”бҹ’бһҡбҹӮ
бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ **бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹҸбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһҸбһ¶бһҳбһ—бһ¶бһҹбһ¶**бҹ”

бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ“бҹҒбҹҮбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷ Co-op Translatorбҹ–

- бһҹбҹ’бһӮбһ¶бһӣбҹӢбһҖбһ¶бһҡбһ”бһҖбһ”бҹ’бһҡбҹӮбһ…бһ¶бһҹбҹӢбһ‘бһ»бһҖбһ”бһ¶бһ“бһҷбһӣбҹӢбһ…бҹ’бһ”бһ¶бһҹбҹӢ
- бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҡбһ¶бһӣбҹӢ Markdown бһҡбһјбһ”бһ—бһ¶бһ– бһ“бһ·бһ„бһҹбҹҖбһңбһ—бҹ…бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһІбҹ’бһҷбһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҺбҹҲбһҹбҹ’бһҡбһҠбҹҖбһ„бһӮбҹ’бһ“бһ¶
- бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ–бһ„бҹ’бһҡбһёбһҖбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһӣбһҫбһғбҹ’бһӣбһ¶бҹҶбһ„бһ’бҹҶбҹ—бһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҷбҹүбһ¶бһ„бһӣбһҝбһ“ бһ“бһ·бһ„бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ—бһ¶бһҹбһ¶бһ…бҹ’бһҡбһҫбһ“

бһҠбҹ„бһҷбһҖбһ¶бһҡбһ•бҹ’бһҸбһӣбҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһ”бһҖбһ”бҹ’бһҡбҹӮбһҮбһ¶бһ‘бҹ’бһҡбһ–бҹ’бһҷбһҹбһҳбҹ’бһ”бһҸбҹ’бһҸбһ·бһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһ¶бһ“
бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһҖбһ”бҹ’бһҡбҹӮбһҹбҹ’бһҡбһ”бһ‘бҹ…бһ“бһ№бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһ—бһ¶бһ–бһ“бһ·бһ„бһ‘бҹ’бһҡбһ–бҹ’бһҷбһҹбһҳбҹ’бһ”бһҸбҹ’бһҸбһ·бһҹбһјбһ бҹ’бһңбһңбҹӮбһҡбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“!

## бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһҳбҹҒбһҡбҹҖбһ“бһҸбһ·бһ…бһҸбһҪбһ…бһ”бҹҶбһ•бһ»бһҸ

1. бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһ¶бһўбҹ’бһ“бһҖбһҳбһ¶бһ“ Python бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҠбҹӮбһӣбһӮбһ¶бҹҶбһ‘бҹ’бһҡ (бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“ 3.10-3.12)бҹ” бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ poetry (pyproject.toml) бһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбһ¶бһҳбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бҹ”
2. бһ”бһ„бҹ’бһҖбһҫбһҸбһҜбһҖбһҹбһ¶бһҡ `.env` бһ”бҹ’бһҡбһҫбһ–бһ»бһҳбҹ’бһ–: [.env.template](../../.env.template)
3. бһҖбҹҶбһҺбһҸбҹӢбһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢ LLM бһҳбһҪбһҷ (Azure OpenAI бһ¬ OpenAI)
4. (бһ”бһҳбҹ’бһҡбһ»бһ„) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһҖбһ”бҹ’бһҡбҹӮбһҡбһјбһ”бһ—бһ¶бһ– (`-img`), бһҖбҹҶбһҺбһҸбҹӢ Azure AI Vision
5. (бһ”бһҳбҹ’бһҡбһ»бһ„) бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҖбҹҶбһҺбһҸбҹӢбһҖбҹҶбһҺбһҸбҹӢбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһҠбҹ„бһҷбһ…бһҳбҹ’бһӣбһ„ бһўбһҗбҹҒбһҡбһҮбһ¶бһҳбһҪбһҷбһҲбһ»бһҸ `_1`, `_2`, бһӣбҹ” бһўбһҗбҹҒбһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһјбһңбһҳбһ¶бһ“бһҲбһ»бһҸбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бҹ”
6. (бһҺбҹӮбһ“бһ¶бҹҶ) бһҹбһҳбҹ’бһўбһ¶бһҸбһҖбһ¶бһҡбһ”бһҖбһ”бҹ’бһҡбҹӮбһ…бһ¶бһҹбҹӢбҹ— бһҠбһҫбһҳбҹ’бһ”бһёбһҸбҹ’бһҡбһјбһңбһӣбһ»бһ”бһ’бҹ’бһңбһҫбһҮбҹҶбһ“бһҪбһҹ (бһ§. `translations/`)
7. (бһҺбҹӮбһ“бһ¶бҹҶ) бһ”бһ“бҹ’бһҗбҹӮбһҳбһ•бҹ’бһ“бҹӮбһҖбһ”бһҖбһ”бҹ’бһҡбҹӮбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„ README бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ [бһ–бһ»бһҳбҹ’бһ–бһ—бһ¶бһҹбһ¶ README](./getting_started/README_languages_template.md)
8. бһҳбһҫбһӣбҹ– [бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„ Azure AI](./getting_started/set-up-azure-ai.md)

- бһҖбһ¶бһҡбһ”бһҖбһ”бҹ’бһҡбҹӮбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Markdown, бһҹбҹҖбһңбһ—бҹ…бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶ бһ“бһ·бһ„бһҡбһјбһ”бһ—бһ¶бһ–
