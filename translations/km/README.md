## бһҹбһ„бҹ’бһҒбҹҒбһ”

**Co-op Translator** бһҮбһҪбһҷбһўбҹ’бһ“бһҖбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһҳбҹ’бһӣбҹӮбһ„бһҳбһ¶бһҸбһ·бһҖбһ¶ GitHub бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһўбһ”бҹӢбһҡбҹҶбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бҹ…бһҮбһ¶бһ—бһ¶бһҹбһ¶бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһҖбһӣбһ·бһ”бһ·бһҖбһҳбҹ’бһҳбҹ”
бһ–бҹҒбһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһҜбһҖбһҹбһ¶бһҡ Markdown бһҡбһјбһ”бһ—бһ¶бһ– бһ¬бһҹбҹҖбһңбһ—бҹ…бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶ бһҖбһ¶бһҡбһ”бҹ’бһҡбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҮбһ¶бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· бһҠбһҫбһҳбҹ’бһ”бһёбһ’бһ¶бһ“бһ¶бһҗбһ¶бһҳбһ¶бһҸбһ·бһҖбһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ“бһ·бһ„бһ‘бһ¶бһ“бҹӢбһҹбһҳбҹҗбһҷ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһҡбҹҖбһ“бһҮбһ¶бһ‘бһјбһ‘бһ¶бҹҶбһ„бһӣбҹ„бһҖбҹ”

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһҳбһ¶бһҸбһ·бһҖбһ¶бһ”бҹ’бһҡбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбҹҖбһ”бһ…бҹҶбҹ–

## бһҡбһ”бҹҖбһ”бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҖбһ¶бһҡбһ”бҹ’бһҡбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣ

Co-op Translator бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҳбһ¶бһҸбһ·бһҖбһ¶бһ”бҹ’бһҡбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһҮбһ¶бһҡбһјбһ”бһҳбһ“бҹ’бһҸбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҖбҹҶбһҺбҹӮ,  
бһҳбһ·бһ“бһҳбҹӮбһ“бһҜбһҖбһҹбһ¶бһҡбһҹбҹ’бһҗбһ·бһҸбһҹбҹ’бһҗбҹҒбһҡбһ“бҹ„бҹҮбһ‘бҹҒбҹ”

бһ§бһ”бһҖбһҡбһҺбҹҚбһ“бҹҒбҹҮбһҸбһ¶бһҳбһҠбһ¶бһ“бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ“бҹғ Markdown бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһҖбһ”бҹ’бһҡбҹӮ бһҡбһјбһ”бһ—бһ¶бһ– бһ“бһ·бһ„бһҹбҹҖбһңбһ—бҹ…бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶  
бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ вҖӢ**metadata бһҠбҹӮбһӣбһҠбһ¶бһҖбҹӢбһҹбҹ’бһҸбһ¶бһҡбһҖбҹ’бһ“бһ»бһ„бһ—бһ¶бһҹбһ¶**бҹ”

бһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһ“бҹҒбҹҮбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷ Co-op Translator:

- бһҡбһҖбһғбһҫбһүбһҖбһ¶бһҡбһ”бҹ’бһҡбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһ…бһ¶бһҹбҹӢбһ‘бһ¶бһ“бҹӢбһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһ”бһ¶бһ“бһҷбһ»бһҸбҹ’бһҸбһ·бһ’бһҳбҹҢ
- бһ”бҹ’бһҡбһҫбһҖбҹ’бһҡбһ»бһҳбһҳбҹүбһ¶бһҡбһј Markdown бһҡбһјбһ”бһ—бһ¶бһ– бһ“бһ·бһ„бһҹбҹҖбһңбһ—бҹ…бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶ бһҠбҹ„бһҷбһ¬бһҸбҹ’бһҡбһјбһңбһӮбҹ’бһ“бһ¶
- бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһӣбһ¶бһҹбҹӢбһҒбҹ’бһҹбҹӮбһҠбҹ„бһҷбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһӣбһҫбһӮбһҳбҹ’бһҡбҹ„бһ„бһ’бҹҶбҹ— бһҠбҹӮбһӣбһҳбһ¶бһ“бһ—бһ¶бһҹбһ¶бһ…бҹ’бһҡбһҫбһ“ бһ“бһ·бһ„бһҡбһ бҹҗбһҹ

бһҠбҹ„бһҷбһӮбҹҶбһҡбһјбһҖбһ¶бһҡбһ”бҹ’бһҡбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһҮбһ¶бһҡбһјбһ”бһҳбһ“бҹ’бһҸбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһ¶бһ“,  
бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бҹ’бһҡбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһҳбҹ’бһҡбһҪбһӣбһҮбһ¶бһҹбҹ’бһҡбһ”бһ‘бҹ…бһ“бһ№бһ„  
бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖ бһ“бһ·бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ‘бҹҶбһ“бһҫбһ”бҹ”

## бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһҸбһ·бһ…бһҸбһҪбһ…

1. бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһ¶бһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҖбҹҶбһҺбҹӮ Python бһҠбҹӮбһӣбһӮбһ¶бҹҶбһ‘бҹ’бһҡ (бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“ 3.10-3.12)бҹ” бһҖбҹ’бһ“бһ»бһ„ poetry (pyproject.toml) бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһ¶бһҸбҹӢбҹ”
2. бһ”бһ„бҹ’бһҖбһҫбһҸбһҜбһҖбһҹбһ¶бһҡ `.env` бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһӮбҹҶбһҡбһјбҹ– [.env.template](../../.env.template)
3. бһҖбҹҶбһҺбһҸбҹӢбһўбҹ’бһ“бһҖбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢ LLM бһҳбһҪбһҷ (Azure OpenAI бһ¬ OpenAI)
4. (бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹ) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбҹӮбһҡбһјбһ”бһ—бһ¶бһ– (`-img`), бһҖбҹҶбһҺбһҸбҹӢ Azure AI Vision
5. (бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹ) бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҖбҹҶбһҺбһҸбҹӢбһҖбҹҶбһҺбһҸбҹӢбһ”бһҺбҹ’бһҠбһ¶бһүбһҹбһүбҹ’бһүбһ¶бһ бҹ’бһӮбҹ„бһӣбһ…бҹ’бһҡбһҫбһ“бһҠбҹ„бһҷбһ…бһҳбҹ’бһӣбһ„бһўбһҗбҹҒбһҡбһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһҮбһҫбһ„бһҸбҹҶбһҺбһҡвҖӢбһҠбһјбһ…бһҮбһ¶ `_1`, `_2`, бһӣбҹ” бһўбһҗбҹҒбһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһјбһңбһҳбһ¶бһ“бһҮбһҫбһ„бһҸбҹҶбһҺбһҡвҖӢбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бҹ”
6. (бһ•бҹ’бһҸбһӣбҹӢбһўбһ“бһ»бһҹбһ¶бһҹбһ“бҹҚ) бһҹбһҳбҹ’бһўбһ¶бһҸбһҖбһ¶бһҡбһ”бҹ’бһҡбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһ…бһ¶бһҹбҹӢбҹ—бһҠбһҫбһҳбҹ’бһ”бһёбһҮбҹҖбһҹбһңбҹҖбһ„бһҖбһ¶бһҡбһ”бҹүбҹҮбһ‘бһ„бҹ’бһӮбһ·бһ… (бһ§. `translations/`)
7. (бһ•бҹ’бһҸбһӣбҹӢбһўбһ“бһ»бһҹбһ¶бһҹбһ“бҹҚ) бһ”бһ“бҹ’бһҗбҹӮбһҳбһ•бҹ’бһ“бҹӮбһҖбһҖбһ¶бһҡбһ”бҹ’бһҡбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„ README бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ [README languages template](./getting_started/README_languages_template.md)
