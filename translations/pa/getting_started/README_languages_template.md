# 🌐 ਬਹੁ ਭਾਸ਼ਾ ਸਹਾਇਤਾ (ਟੈਮਪਲੇਟ)

ਮੈਨਟੇਨਰ: ਹੇਠਾਂ ਦਿੱਤਾ ਬਲਾਕ Co-op Translator ਵੱਲੋਂ ਸੰਭਾਲਿਆ ਗਿਆ "ਸਭ ਭਾਸ਼ਾਵਾਂ" ਦਾ ਉਦਾਹਰਨ ਹੈ।

- ਜੇ ਤੁਸੀਂ ਚਾਹੁੰਦੇ ਹੋ ਕਿ Co-op Translator ਇਸ ਸੂਚੀ ਨੂੰ ਪੂਰੀ ਤਰ੍ਹਾਂ ਤੁਹਾਡੇ `translate` (ਕਿਸੇ ਵੀ ਭਾਸ਼ਾ ਚੋਣ) ਚਲਾਉਣ 'ਤੇ ਆਪੋ-ਆਪਦੇ ਤਰੀਕੇ ਨਾਲ ਅਪਡੇਟ ਕਰਦਾ ਰਹੇ, ਤਾਂ ਦੋਨੋਂ ਟਿੱਪਣੀਆਂ ਦੇ ਨਿਸ਼ਾਨਾਂ ਨੂੰ ਜਿਵੇਂ ਹਨ ਤਿਵੇਂ ਰੱਖੋ।
- ਜੇ ਤੁਸੀਂ ਸਿਰਫ ਕੁਝ ਭਾਸ਼ਾਵਾਂ ਦਿਖਾਉਣਾ ਚਾਹੁੰਦੇ ਹੋ, ਤਾਂ ਦੋਨੋਂ ਟਿੱਪਣੀਆਂ ਦੇ ਨਿਸ਼ਾਨਾਂ ਨੂੰ ਮਿਟਾ ਦਿਓ ਅਤੇ ਉਹ ਭਾਸ਼ਾਵਾਂ ਹਟਾ ਦਿਓ ਜਿਹੜੀਆਂ ਤੁਸੀਂ ਸੂਚੀ ਵਿੱਚ ਨਹੀਂ ਚਾਹੁੰਦੇ। ਟਿੱਪਣੀਆਂ ਦੇ ਨਿਸ਼ਾਨ ਹਟਾਉਣ ਤੋਂ ਬਾਅਦ, Co-op Translator ਹੁਣ ਇਸ ਹਿੱਸੇ ਨੂੰ ਆਟੋ-ਰਿਪਲੇਸ ਨਹੀਂ ਕਰੇਗਾ।

- ਇਸ ਹਿੱਸੇ ਵਿੱਚ ਹੁਣ ਇੱਕ "ਲੋਕਲ ਤੌਰ ਤੇ ਕਲੋਨ ਕਰਨਾ ਪਸੰਦ ਹੈ?" ਸਲਾਹ ਸ਼ਾਮਿਲ ਹੈ, ਜੋ ਉਪਭੋਗਤਾਵਾਂ ਨੂੰ ਵੱਡੇ ਅਨੁਵਾਦ ਪੇਲੋਡ ਦੇ ਬਿਨਾ ਕਲੋਨ ਕਰਨ ਵਿੱਚ ਮਦਦ ਕਰਦਾ ਹੈ। ਤੁਸੀਂ ਆਪਣੇ ਰਿਪੋਜ਼ਟਰੀ URL ਨਾਲ ਸਲਾਹ ਨੂੰ ਵਿਆਕਤੀਗਤ ਕਰ ਸਕਦੇ ਹੋ, ਉਦਾਹਰਨ ਵਜੋਂ:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
