# 🌐 ਬਹੁ-ਭਾਸ਼ਾਈ ਸਹਾਇਤਾ (ਟੈਂਪਲੇਟ)

ਰਖਿਆ ਕਰਨ ਵਾਲੇ: ਹੇਠਾਂ ਦਿੱਤਾ ਬਲਾਕ Co-op Translator ਵੱਲੋਂ ਸੰਭਾਲਿਆ ਗਿਆ "ਸਭ ਭਾਸ਼ਾਵਾਂ" ਉਦਾਹਰਨ ਹੈ।

- ਜੇ ਤੁਸੀਂ ਚਾਹੁੰਦੇ ਹੋ ਕਿ ਜਦੋਂ ਤੁਸੀਂ `translate` ਚਲਾਉਂਦੇ ਹੋ (ਕਿਸੇ ਵੀ ਭਾਸ਼ਾ ਚੋਣ ਲਈ) Co-op Translator ਇਸ ਸੂਚੀ ਨੂੰ ਪੂਰੀ ਤਰ੍ਹਾਂ ਆਪਣੇ ਆਪ ਅਪ-ਟੂ-ਡੇਟ ਰੱਖੇ, ਤਾਂ ਦੋਨੋ ਟਿੱਪਣੀਆਂ ਵਾਲੇ ਨਿਸ਼ਾਨ ਉਨ੍ਹਾਂ ਹੀ ਤਰ੍ਹਾਂ ਛੱਡੋ।
- ਜੇ ਤੁਸੀਂ ਕੇਵਲ ਭਾਸ਼ਾਵਾਂ ਦੇ ਇੱਕ ਉਪਸੈੱਟ ਨੂੰ ਦਿਖਾਉਣਾ ਚਾਹੁੰਦੇ ਹੋ, ਤਾਂ ਦੋਨੋ ਟਿੱਪਣੀਆਂ ਵਾਲੇ ਨਿਸ਼ਾਨ ਹਟਾ ਦਿਓ ਅਤੇ ਉਹ ਭਾਸ਼ਾਵਾਂ ਹਟਾ ਦਿਓ ਜੋ ਤੁਸੀਂ ਸੂਚੀ ਵਿੱਚ ਨਹੀਂ ਚਾਹੁੰਦੇ। ਨਿਸ਼ਾਨ ਹਟਾਉਣ ਤੋਂ ਬਾਅਦ, Co-op Translator ਹੁਣ ਇਸ ਸੈਕਸ਼ਨ ਨੂੰ ਆਪਣੇ ਆਪ ਬਦਲਣ ਨਹੀਂ ਕਰੇਗਾ।

- ਹੁਣ ਸੈਕਸ਼ਨ ਵਿੱਚ "स्थानीय तौर 'ਤੇ ਕਲੋਨ ਕਰਨਾ ਪਸੰਦ ਕਰੋ?" ਸਲਾਹ ਸ਼ਾਮਲ ਹੈ ਤਾਂ ਜੋ ਉਪਭੋਗਤਾਵਾਂ ਵੱਡੇ ਅਨੁਵਾਦ payload ਤੋਂ ਬਿਨਾਂ ਕਲੋਨ ਕਰ ਸਕਣ। ਤੁਸੀਂ ਆਪਣੀ ਰਿਪੋਜ਼ਟਰੀ URL ਨਾਲ ਇਸ ਸਲਾਹ ਨੂੰ ਵਿਅਕਤਿਗਤ ਕਰ ਸਕਦੇ ਹੋ, ਉਦਾਹਰਨ ਲਈ:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
