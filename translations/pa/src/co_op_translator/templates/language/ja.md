# Localizeflow ਵਿੱਚ ਇੱਕ ਪਰਿਵਰਤਨ ਯੋਗ GitHub App

Localizeflow ਹੁਣ ਇੱਕ ਪਰਿਵਰਤਨ ਯੋਗ GitHub App ਦੇ ਤੌਰ 'ਤੇ ਉਪਲਬਧ ਹੈ, ਜਿਸ ਤੋਂ ਤੁਹਾਨੂੰ ਫਾਇਦਾ ਮਿਲਦਾ ਹੈ:

- ਕੋਈ ਸਥਾਨਕ ਸੈਟਅੱਪ ਦੀ ਲੋੜ ਨਹੀਂ ਹੈ.
- ਤੁਸੀਂ ਆਪਣੇ GitHub ਗਠਜੋੜ ਦੇ ਸਾਡੇ ਨਾਲ ਇਕੱਠੇ ਆਪਝੜੇ ਹੋ.
- ਖਾਸ ਪ੍ਰੋਜੈਕਟਾਂ ਲਈ ਅਲੱਗ ਅਧਿਕਾਰ ਸੈਟ ਕਰੋ.
- ਇੱਕ-ਕਿਲਕ ਇੰਸਟਾਲੇਸ਼ਨ ਪ੍ਰਕਿਰਿਆ ਤੇਜ਼ ਅਤੇ ਆਸਾਨ ਬਨਾਓ.

## Localizeflow ਨੂੰ ਆਪਣੇ GitHub ਗਠਜੋੜ ਵਿੱਚ ਕਿਵੇਂ ਜੋੜੀਏ

1. ਆਪਣੇ ਪ੍ਰੋਜੈਕਟ ਲਈ [Localizeflow GitHub App](https://github.com/apps/localizeflow) ਤੇ ਜਾਓ ਅਤੇ "ਇੰਸਟਾਲ ਕਰੋ" ਉੱਤੇ ਕਲਿਕ ਕਰੋ.
2. ਉਹ ਗਿਟਹਬ ਗਠਜੋੜ ਚੁਣੋ ਜਿੱਥੇ ਤੁਹਾਡੇ ਦਾੁਬਾਰਾ ਅਨੁਵਾਦ/ਲੋਕਲਾਈਜੇਸ਼ਨ ਪ੍ਰਕਿਰਿਆ ਹੈ.
3. ਅਸੈੱਸ ਪਸੰਦਗੀਆਂ ਸੈਟ ਕਰੋ (ਜਿਵੇਂ ਕਿ ਸਿਰਫ਼ ਕਿਛੇ ਸਟੇਟਸਾਂ ਅਤੇ ਫਾਈਲਾਂ ਲਈ ਅਨੁਵਾਦ).
4. ਇੰਸਟਾਲੇਸ਼ਨ ਨੂੰ ਪੁਸ਼ਟੀਤ ਕਰੋ.

## Localizeflow GitHub App ਦੀ ਵਰਤੋਂ ਕਰਨ ਦਾ ਫਾਇਦਾ

- ਤੁਹਾਡਾ ਮਨੁੱਖੀ ਅਨੁਵਾਦਕ ਪ੍ਰਕਿਰਿਆ ਸਹੀ ਢੰਗ ਨਾਲ ਖੁੱਲੇ GitHub ਪਰਿਵਰਤਨਾਂ ਅਤੇ ਪੁਲ ਰਿਕਵੇਸਟਸ ਵਿੱਚ ਇੰਟੀਗ੍ਰੇਟ ਹੁੰਦੀ ਹੈ.
- ਅਨੁਵਾਦ ਫਾਈਲਾਂ ਦੀ ਸਥਾਨਕ ਪਰਿਵਰਤਨ ਯੋਗਤਾ ਨੂੰ ਸਰਲ ਬਣਾਉਦਾ ਹੈ.
- ਕੋਅਪ ਟਰਾਂਸਲੇਟਰ ਲਈ ਸਹਿਯੋਗੀ ਅਨੁਵਾਦ ਦੇ ਸਰੋਤ 'ਤੇ ਪੂਰੀ ਨਿਗਰਾਨੀ ਪ੍ਰਦਾਨ ਕਰਦਾ ਹੈ.
- ਜੇਕਰ ਤੁਹਾਡਾ ਪ੍ਰੋਜੈਕਟ ਪਹਿਲਾਂ ਤਰ੍ਹਾਂ ਸੈੱਟ ਕੀਤਾ ਹੈ, ਤਾਂ Localizeflow ਤੁਹਾਡੇ ਲਈ ਹਰ ਇੱਕ ਪੂਲ ਰਿਕਵੇਸਟ ਦੇ ਤੌਰ 'ਤੇ ਅਨੁਵਾਦ ਤਿਆਰ ਕਰੇਗਾ. 

## ਅੱਗੇ ਕੀ?

ਤੁਹਾਡੇ ਟੀਮ ਲਈ Localizeflow GitHub App ਨੂੰ ਵਰਤਣਾ ਸ਼ੁਰੂ ਕਰਨ ਲਈ, ਸਿਰਫ ਉਪਰੋਕਤ ਚਰਣਾਂ ਨੂੰ ਪ徍ਰ ਕਰੋ. ਅਸੀਂ ਤੁਹਾਡੇ ਫੀਡਬੈਕ ਅਤੇ ਸੁਝਾਵਾਂ ਲਈ ਉਤਸ਼ਾਹਤ ਹਾਂ!

ਕੀ ਤੁਹਾਡੇ ਕੋਲ ਹੋਰ ਸਵਾਲ ਹਨ?ਸਾਨੂੰ [GitHub ਪਰੋਜੈਕਟ ਰੀਪੋ](https://github.com/Azure/localizeflow) 'ਤੇ ਕਰੋ ਜਾਂ ਸਾਡੇ [ਸਮਰਥਨ ਚੈਨਲ](https://aka.ms/localizeflow-support) ਵਿੱਚ ਸ਼ਾਮਲ ਹੋਵੋ।
