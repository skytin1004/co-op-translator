>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse <repo_url>.git
> cd <repo_name>
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse <repo_url>.git
> cd <repo_name>
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> ਇਹ ਤੁਹਾਨੂੰ ਕੋਰਸ ਨੂੰ مکمل ਕਰਨ ਲਈ ਸਾਰੀ ਲੋੜੀਂਦੀ ਚੀਜ਼ ਦੇਵੇਗਾ ਤੇ ਇਕ ਤੇਜ਼ ਡਾਊਨਲੋਡ ਵਜੋਂ ਮਿਲੇਗਾ।
