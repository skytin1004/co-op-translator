# 🌐 Daugiakalbė palaikymas (šablonas)

Prižiūrėtojai: žemiau esantis blokas yra „visų kalbų“ pavyzdys, kurį valdo Co-op Translator.

- Jei norite, kad Co-op Translator automatiškai palaikytų šį sąrašą visiškai atnaujintą, kai paleidžiate `translate` (bet kuri kalbos parinktis), palikite abu komentarų žymes kaip yra.
- Jei norite rodyti tik kalbų poaibį, ištrinkite abi komentarų žymes ir pašalinkite kalbas, kurių nenorite sąraše. Pašalinus žymes, Co-op Translator nebeautomatiškai keis šią sekciją.

- Dabar skiltyje yra pranešimas „Ar norite klonuoti lokaliai?“, kuris padeda vartotojams klonuoti be didelio vertimų duomenų kiekio. Galite suasmeninti pranešimą su savo saugyklos URL, pavyzdžiui, vykdydami:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
