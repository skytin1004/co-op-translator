# Palaikomos kalbos

Žemiau pateiktoje lentelėje išvardytos kalbos, kurias šiuo metu palaiko **Co-op Translator**. Joje yra kalbos kodai, kalbų pavadinimai ir žinomos problemos, susijusios su kiekviena kalba. Jei norite pridėti naują kalbą, pridėkite atitinkamą kalbos kodą, pavadinimą ir tinkamą šriftą faile `font_language_mappings.yml`, esančiame `src/co_op_translator/fonts/`, ir pateikite pull request po testavimo.
