# معاون زبانیں

ذیل کی میز میں وہ زبانیں درج ہیں جو فی الحال **Co-op Translator** کی طرف سے معاونت یافتہ ہیں۔ یہ زبان کوڈز، زبانوں کے نام، اور ہر زبان سے متعلق ممکنہ مسائل کو شامل کرتی ہے۔ اگر آپ کسی نئی زبان کو شامل کرنا چاہتے ہیں تو براہ کرم متعلقہ زبان کوڈ، نام، اور مناسب فونٹ فائل `font_language_mappings.yml` میں شامل کریں جو کہ `src/co_op_translator/fonts/` میں واقع ہے اور جانچ پڑتال کے بعد ایک pull request جمع کرائیں۔
