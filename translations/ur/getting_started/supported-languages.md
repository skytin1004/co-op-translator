# زبانیں جن کی معاونت کی جاتی ہے

ذیل کی جدول میں وہ زبانیں درج ہیں جن کی **Co-op Translator** فی الحال حمایت کرتا ہے۔ اس میں زبان کے کوڈ، زبان کے نام، اور ہر زبان سے متعلق کوئی معروف مسائل شامل ہیں۔ اگر آپ کوئی نئی زبان شامل کرنا چاہتے ہیں، تو براہ کرم متعلقہ زبان کا کوڈ، نام، اور مناسب فونٹ `font_language_mappings.yml` فائل میں شامل کریں جو `src/co_op_translator/fonts/` میں موجود ہے اور جانچ کے بعد ایک pull request جمع کروائیں۔
