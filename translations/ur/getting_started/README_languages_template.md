# 🌐 کثیراللسانی معاونت (سانچہ)

منتظمین: نیچے دیا گیا بلاک Co-op Translator کے ذریعہ منظم کردہ "تمام زبانوں" کی مثال ہے۔

- اگر آپ چاہتے ہیں کہ Co-op Translator جب بھی آپ `translate` (کسی بھی زبان کا انتخاب) چلائیں تو اس فہرست کو مکمل طور پر تازہ ترین رکھے، تو دونوں تبصرہ مارکرز کو جوں کا توں رکھیں۔
- اگر آپ صرف زبانوں کے ایک ذیلی سیٹ کو دکھانا چاہتے ہیں، تو دونوں تبصرہ مارکرز حذف کردیں اور وہ زبانیں نکال دیں جنہیں آپ فہرست میں شامل نہیں کرنا چاہتے۔ مارکرز کو ہٹانے کے بعد، Co-op Translator اس سیکشن کو خودکار طور پر تبدیل نہیں کرے گا۔

- اس سیکشن میں اب "کیا مقامی طور پر کلون کرنا پسند کریں گے؟" کی نصیحت شامل ہے تاکہ صارفین کو بڑی ترجمہ شدہ فائل کے بغیر کلون کرنے میں مدد ملے۔ آپ اپنی ذخیرہ URL کے ساتھ اس نصیحت کو حسب خواہش بنا سکتے ہیں، مثال کے طور پر:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
