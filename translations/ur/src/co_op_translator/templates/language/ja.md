# Localizeflow CLI

Localizeflow CLIを使うと、GitHubリポジトリの翻訳ワークフローを管理できます。

## インストール

```bash
npm install -g @localizeflow/cli
```

## 使い方

```bash
localizeflow setup
```

このコマンドはリポジトリを初期化し、[GitHub App](https://github.com/apps/localizeflow)をサポートします。

翻訳プルリクエストを作成するには:

```bash
localizeflow create-pr
```

詳細については[ドキュメント](https://localizeflow.dev/docs)を参照してください。

## トラブルシューティング

問題が発生した場合は、[GitHub Issues](https://github.com/your-repo/issues)で報告してください。
