# 🌐 תמיכה בריבוי שפות (תבנית)

המנהלים: החסימה למטה היא דוגמה ל"כל השפות" המנוהלת על ידי Co-op Translator.

- אם ברצונך ש-Co-op Translator ישמור על רשימה זו מעודכנת באופן אוטומטי בכל פעם שתריץ `translate` (בחירת כל שפה), שמור את שני סימני ההערה כפי שהם.
- אם אתה מעוניין להציג רק תת-קבוצה של שפות, מחק את שני סימני ההערה והסר כל שפה שאינך מעוניין להציג ברשימה. לאחר הסרת סימני ההערה, Co-op Translator לא יחליף עוד אוטומטית מקטע זה.

- הקטע כולל כעת אזהרה של "מעדיפים לשכפל מקומית?" כדי לסייע למשתמשים לשכפל ללא כמות גדולה של תרגומים. תוכל להתאים אישית את ההודעה עם כתובת המאגר שלך על ידי הרצת, לדוגמה:  
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
