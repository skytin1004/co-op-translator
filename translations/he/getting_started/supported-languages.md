# שפות נתמכות

הטבלה למטה מפרטת את השפות הנתמכות כיום על ידי **Co-op Translator**. היא כוללת קודי שפה, שמות שפות וכל בעיה ידועה הקשורה לכל שפה. אם ברצונכם להוסיף תמיכה לשפה חדשה, אנא הוסיפו את קוד השפה המתאים, השם, והגופן המתאים בקובץ `font_language_mappings.yml` הנמצא בנתיב `src/co_op_translator/fonts/` והגישו pull request לאחר בדיקה.
