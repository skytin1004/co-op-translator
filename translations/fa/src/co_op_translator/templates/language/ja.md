# آموزش (پیش‌نمایش)

Localizeflow فرایندی است که با هدف ساده‌سازی و افزایش بهره‌وری در ترجمه همکارانه ایجاد شده است. این مستند گام‌های اصلی برای استفاده از این سیستم را شرح می‌دهد.

## مراحل استفاده از Localizeflow

1. مخزن GitHub خود را به Localizeflow متصل کنید.
2. با استفاده از GitHub App مربوطه، سیستم ترجمه خودکار را فعال کنید.
3. یک PR جدید ایجاد کنید و متن جدید یا اصلاح شده خود را به پروژه اضافه کنید.
4. پس از ایجاد PR، این سیستم متن را به سرویس ترجمه محلی ارسال می‌کند.
5. ترجمه‌ها در قالب کامنت در همان PR ظاهر خواهند شد.
6. ترجمه‌های ارائه شده را بررسی و در صورت نیاز ویرایش کنید.
7. پس از تأیید، ترجمه‌ها به صورت خودکار به شاخه شما ادغام می‌شوند.

به این ترتیب، می‌توانید به راحتی محتوای خود را همزمان به چند زبان توسعه دهید و همکاری تیمی را بهینه کنید.

برای جزئیات بیشتر به مستندات [Co-op Translator](https://github.com/Azure/co-op-translator) مراجعه کنید.
