مد ژاپنی: به شدت نشانه‌گذاری‌های Markdown را حفظ کنید.

قوانین (باید رعایت شوند):
1) لینک‌های Markdown را دقیقاً نگه دارید: [text](../../../../../../src/co_op_translator/templates/language/URL) -> [متن ترجمه شده](../../../../../../src/co_op_translator/templates/language/همان URL).
2) هرگز لینک‌ها را به صورت متن ساده بازنویسی نکنید (مثلاً 「text」（URL）، text (URL)).
3) فقط متن لینک را ترجمه کنید؛ ساختار Markdown و URL را تغییر ندهید.
4) دور یک لینک Markdown 「」 اضافه نکنید مگر اینکه دستور زبان خارج از لینک آن را ایجاب کند.

ساختار مهم‌تر از سبک است.
اگر قرار است نشانه‌گذاری‌های Markdown تغییر کنند، طبیعی بودن ژاپنی را بهینه نکنید.

مثال
منبع: This document uses [Co-op Translator](https://github.com/Azure/co-op-translator).
صحیح: 本書類は [Co-op Translator](https://github.com/Azure/co-op-translator) を使用しています。
نادرست: 本書類は「Co-op Translator」（https://github.com/Azure/co-op-translator）を使用しています。
