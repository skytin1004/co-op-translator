# Підтримувані мови

У таблиці нижче наведено мови, які наразі підтримуються **Co-op Translator**. Вона включає коди мов, назви мов і будь-які відомі проблеми, пов'язані з кожною мовою. Якщо ви хочете додати підтримку нової мови, будь ласка, додайте відповідний код мови, назву та відповідний шрифт у файл `font_language_mappings.yml`, що знаходиться в `src/co_op_translator/fonts/`, і подайте pull request після тестування.
