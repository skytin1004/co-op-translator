# 🌐 Поддръжка на многоезичност (Шаблон)

Поддържачи: Блокът по-долу е пример за "всички езици", управляван от Co-op Translator.

- Ако искате Co-op Translator да поддържа този списък напълно актуален автоматично, когато стартирате `translate` (избор на всеки език), запазете двата коментара непроменени.
- Ако искате да показвате само подмножество от езици, изтрийте двата коментара и премахнете езиците, които не искате да включвате. След премахване на коментарите, Co-op Translator няма да обновява автоматично този раздел.

- Разделът вече включва съвет "Предпочитате ли да клонирате локално?", който помага на потребителите да клонират без големия обем с преводи. Можете да персонализирате този съвет с URL към вашето репо, като например:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
