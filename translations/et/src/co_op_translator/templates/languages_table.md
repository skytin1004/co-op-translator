>
> See hoidla sisaldab üle 50 keele tõlget, mis suurendab märkimisväärselt allalaadimise mahtu. Tõlgeteta kloonimiseks kasuta hõredat checkout’i:
>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse <repo_url>.git
> cd <repo_name>
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse <repo_url>.git
> cd <repo_name>
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> See annab sulle kõik vajaliku kursuse läbimiseks palju kiirema allalaadimisega.
