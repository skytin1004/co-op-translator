# 🌐 Mitmekeelne tugi (Mall)

Hooldajad: allolev plokk on "kõik keeled" näide, mida haldab Co-op Translator.

- Kui soovite, et Co-op Translator hoiab selle nimekirja automaatselt täielikult ajakohasena, kui käivitate `translate` (suvalise keele valiku korral), jätke kaks kommentaari märgistust nii, nagu need on.
- Kui soovite näidata ainult keelte alamhulka, kustutage kaks kommentaari märgistust ning eemaldage soovimatud keeled. Märgistuste eemaldamisel ei asenda Co-op Translator enam seda osa automaatselt.

- Sektsioon sisaldab nüüd soovitust „Eelistate kloonida lokaalselt?“, et aidata kasutajatel kloonida ilma suure tõlkemahuta. Võite soovitust personaliseerida oma repositooriumi URL-iga, käivitades näiteks:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
