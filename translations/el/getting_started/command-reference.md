# Command reference

Το CLI του **Co-op Translator** προσφέρει πολλές επιλογές για την προσαρμογή της διαδικασίας μετάφρασης:

Command                                       | Περιγραφή
----------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
translate -l "language_codes"                 | Μεταφράζει το έργο σας στις καθορισμένες γλώσσες. Παράδειγμα: translate -l "es fr de" μεταφράζει στα Ισπανικά, Γαλλικά και Γερμανικά. Χρησιμοποιήστε translate -l "all" για μετάφραση σε όλες τις υποστηριζόμενες γλώσσες.
translate -l "language_codes" -u              | Ενημερώνει τις μεταφράσεις διαγράφοντας τις υπάρχουσες και επαναδημιουργώντας τες. Προσοχή: Αυτό θα διαγράψει όλες τις τρέχουσες μεταφράσεις για τις καθορισμένες γλώσσες.
translate -l "language_codes" -img            | Μεταφράζει μόνο αρχεία εικόνων.
translate -l "language_codes" -md             | Μεταφράζει μόνο αρχεία Markdown.
translate -l "language_codes" -nb             | Μεταφράζει μόνο αρχεία Jupyter notebook (.ipynb).
translate -l "language_codes" --fix           | Επαναμεταφράζει αρχεία με χαμηλούς δείκτες εμπιστοσύνης βάσει προηγούμενων αποτελεσμάτων αξιολόγησης.
translate -l "language_codes" -d              | Ενεργοποιεί τη λειτουργία αποσφαλμάτωσης για λεπτομερή καταγραφή.
translate -l "language_codes" --save-logs, -s | Αποθηκεύει αρχεία καταγραφής DEBUG στο <root_dir>/logs/ (η κονσόλα παραμένει ελεγχόμενη από -d)
translate -l "language_codes" -r "root_dir"   | Καθορίζει τον ριζικό φάκελο του έργου
translate -l "language_codes" -f              | Χρησιμοποιεί γρήγορη λειτουργία για μετάφραση εικόνων (έως 3 φορές ταχύτερο σχεδιασμό με ελαφριά απόκλιση ποιότητας και στοίχισης).
translate -l "language_codes" -y              | Επιβεβαιώνει αυτόματα όλα τα μηνύματα (χρήσιμο για CI/CD pipelines)
translate -l "language_codes" --add-disclaimer/--no-disclaimer | Ενεργοποιεί ή απενεργοποιεί την προσθήκη ρήτρας μηχανικής μετάφρασης στα μεταφρασμένα αρχεία markdown και notebook (προεπιλογή: ενεργοποιημένη).
translate -l "language_codes" --repo-url "https://github.com/org/repo.git" | Εξατομικεύει την ενότητα συμβουλών για τις γλώσσες στο README (sparse checkout) με το URL του αποθετηρίου σας.
translate -l "language_codes" --help          | λεπτομέρειες βοήθειας στο CLI που δείχνουν διαθέσιμες εντολές
evaluate -l "language_code"                  | Αξιολογεί την ποιότητα μετάφρασης για μια συγκεκριμένη γλώσσα και παρέχει δείκτες εμπιστοσύνης
evaluate -l "language_code" -c 0.8           | Αξιολογεί μεταφράσεις με προσαρμοσμένο όριο εμπιστοσύνης
evaluate -l "language_code" -f               | Γρήγορη λειτουργία αξιολόγησης (μόνο βασισμένη σε κανόνες, χωρίς LLM)
evaluate -l "language_code" -D               | Βαθιά λειτουργία αξιολόγησης (μόνο βασισμένη σε LLM, πιο λεπτομερής αλλά πιο αργή)
evaluate -l "language_code" --save-logs, -s  | Αποθηκεύει αρχεία καταγραφής DEBUG στο <root_dir>/logs/
migrate-links -l "language_codes"             | Επεξεργάζεται ξανά μεταφρασμένα αρχεία Markdown για ενημέρωση συνδέσμων σε notebooks (.ipynb). Προτιμά μεταφρασμένα notebooks όταν υπάρχουν· αλλιώς μπορεί να επιστρέψει στα πρωτότυπα.
migrate-links -l "language_codes" -r          | Καθορίζει τον ριζικό φάκελο του έργου (προεπιλογή: τρέχων κατάλογος).
migrate-links -l "language_codes" --dry-run   | Δείχνει ποια αρχεία θα αλλάξουν χωρίς να γράψει αλλαγές.
migrate-links -l "language_codes" --no-fallback-to-original | Μην ξαναγράφει συνδέσμους προς πρωτότυπα notebooks όταν λείπουν μεταφρασμένα αντίγραφα (ενημερώνει μόνο όταν υπάρχει μετάφραση).
migrate-links -l "language_codes" -d          | Ενεργοποιεί λειτουργία αποσφαλμάτωσης για λεπτομερή καταγραφή.
migrate-links -l "language_codes" --save-logs, -s | Αποθηκεύει αρχεία καταγραφής DEBUG στο <root_dir>/logs/
migrate-links -l "all" -y                      | Επεξεργάζεται όλες τις γλώσσες και επιβεβαιώνει αυτόματα το μήνυμα προειδοποίησης.

## Usage examples

  1. Προεπιλεγμένη συμπεριφορά (προσθήκη νέων μεταφράσεων χωρίς διαγραφή των υπαρχουσών):   translate -l "ko"    translate -l "es fr de" -r "./my_project"

  2. Προσθήκη μόνο νέων μεταφράσεων εικόνων για Κορεάτικα (χωρίς διαγραφή υπαρχουσών μεταφράσεων):    translate -l "ko" -img

  3. Ενημέρωση όλων των μεταφράσεων Κορεάτικων (Προσοχή: Αυτό διαγράφει όλες τις υπάρχουσες μεταφράσεις Κορεάτικων πριν την επαναμετάφραση):    translate -l "ko" -u

  4. Ενημέρωση μόνο εικόνων Κορεάτικων (Προσοχή: Αυτό διαγράφει όλες τις υπάρχουσες εικόνες Κορεάτικων πριν την επαναμετάφραση):    translate -l "ko" -img -u

  5. Προσθήκη νέων μεταφράσεων markdown για Κορεάτικα χωρίς να επηρεάσει άλλες μεταφράσεις:    translate -l "ko" -md

  6. Επιδιόρθωση μεταφράσεων με χαμηλή εμπιστοσύνη βάσει προηγούμενων αποτελεσμάτων αξιολόγησης: translate -l "ko" --fix

  7. Επιδιόρθωση μεταφράσεων με χαμηλή εμπιστοσύνη μόνο για συγκεκριμένα αρχεία (markdown): translate -l "ko" --fix -md

  8. Επιδιόρθωση μεταφράσεων με χαμηλή εμπιστοσύνη μόνο για συγκεκριμένα αρχεία (εικόνες): translate -l "ko" --fix -img

  9. Χρήση γρήγορης λειτουργίας για μετάφραση εικόνων:    translate -l "ko" -img -f

  10. Επιδιόρθωση μεταφράσεων με χαμηλή εμπιστοσύνη με προσαρμοσμένο όριο: translate -l "ko" --fix -c 0.8

  11. Παράδειγμα λειτουργίας αποσφαλμάτωσης: - translate -l "ko" -d: Ενεργοποίηση καταγραφής αποσφαλμάτωσης.
  12. Αποθήκευση καταγραφών σε αρχεία: translate -l "ko" -s
  13. DEBUG στην κονσόλα και σε αρχείο: translate -l "ko" -d -s
  14. Μετάφραση χωρίς προσθήκη ρητρών μηχανικής μετάφρασης στα αποτελέσματα: translate -l "ko" --no-disclaimer

  15. Μεταφορά συνδέσμων notebook για μεταφράσεις Κορεάτικων (ενημέρωση συνδέσμων σε μεταφρασμένα notebooks όταν υπάρχουν):    migrate-links -l "ko"

  15. Μεταφορά συνδέσμων με dry-run (χωρίς εγγραφές αρχείων):    migrate-links -l "ko" --dry-run

  16. Ενημέρωση συνδέσμων μόνο όταν υπάρχουν μεταφρασμένα notebooks (χωρίς επιστροφή σε πρωτότυπα):    migrate-links -l "ko" --no-fallback-to-original

  17. Επεξεργασία όλων των γλωσσών με μήνυμα επιβεβαίωσης:    migrate-links -l "all"

  18. Επεξεργασία όλων των γλωσσών με αυτόματη επιβεβαίωση:    migrate-links -l "all" -y
  19. Αποθήκευση καταγραφών για migrate-links:    migrate-links -l "ko ja" -s

  20. Εξατομίκευση συμβουλών για τις γλώσσες στο README με το URL του αποθετηρίου σας:
      translate -l "ko" --repo-url "https://github.com/org/repo.git"

### Evaluation Examples

> [!WARNING]  
> **Beta Feature**: Η λειτουργία αξιολόγησης βρίσκεται προς το παρόν σε beta. Αυτή η δυνατότητα κυκλοφόρησε για την αξιολόγηση μεταφρασμένων εγγράφων, και οι μέθοδοι αξιολόγησης και η λεπτομερής υλοποίηση βρίσκονται ακόμη σε εξέλιξη και ενδέχεται να αλλάξουν.

  1. Αξιολόγηση μεταφράσεων Κορεάτικων: evaluate -l "ko"

  2. Αξιολόγηση με προσαρμοσμένο όριο εμπιστοσύνης: evaluate -l "ko" -c 0.8

  3. Γρήγορη αξιολόγηση (μόνο βασισμένη σε κανόνες): evaluate -l "ko" -f

  4. Βαθιά αξιολόγηση (μόνο βασισμένη σε LLM): evaluate -l "ko" -D
