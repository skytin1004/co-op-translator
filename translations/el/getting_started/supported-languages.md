# Υποστηριζόμενες γλώσσες

Ο παρακάτω πίνακας αναφέρει τις γλώσσες που υποστηρίζονται επί του παρόντος από το **Co-op Translator**. Περιλαμβάνει κωδικούς γλωσσών, ονόματα γλωσσών και τυχόν γνωστά προβλήματα που σχετίζονται με κάθε γλώσσα. Εάν θέλετε να προσθέσετε υποστήριξη για μια νέα γλώσσα, προσθέστε τον αντίστοιχο κωδικό γλώσσας, το όνομα και τη σωστή γραμματοσειρά στο αρχείο `font_language_mappings.yml` που βρίσκεται στο `src/co_op_translator/fonts/` και υποβάλετε ένα pull request μετά τη δοκιμή.
