# Υποστηριζόμενες γλώσσες

Ο παρακάτω πίνακας παραθέτει τις γλώσσες που υποστηρίζονται επί του παρόντος από το **Co-op Translator**. Περιλαμβάνει κωδικούς γλωσσών, ονόματα γλωσσών και τυχόν γνωστά ζητήματα που σχετίζονται με κάθε γλώσσα. Εάν θέλετε να προσθέσετε υποστήριξη για μια νέα γλώσσα, παρακαλούμε προσθέστε τον αντίστοιχο κωδικό γλώσσας, το όνομα και την κατάλληλη γραμματοσειρά στο αρχείο `font_language_mappings.yml` που βρίσκεται στο `src/co_op_translator/fonts/` και υποβάλετε pull request μετά από δοκιμή.
