Mod Jepun: kekalkan token Markdown dengan ketat.

Peraturan (mesti ikut):
1) Kekalkan pautan Markdown dengan tepat: [teks](../../../../../../src/co_op_translator/templates/language/URL) -> [teks terjemahan](../../../../../../src/co_op_translator/templates/language/URL yang sama).
2) JANGAN sesekali menulis pautan sebagai teks biasa (contohnya, 「teks」（URL）, teks (URL)).
3) Terjemah hanya teks pautan; kekalkan struktur Markdown dan URL tanpa perubahan.
4) Jangan tambah 「」 di sekitar pautan Markdown melainkan tatabahasa luar pautan memerlukannya.

STRUKTUR LEBIH PENTING DARIPADA GAYA.
Jangan memperbaiki kesemulajadian Jepun jika token Markdown akan berubah.
