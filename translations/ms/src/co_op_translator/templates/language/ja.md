Mod Bahasa Jepun: kekalkan token Markdown dengan ketat.

Peraturan (wajib ikut):
1) Kekalkan pautan Markdown dengan tepat: [teks](../../../../../../src/co_op_translator/templates/language/URL) -> [teks terjemahan](../../../../../../src/co_op_translator/templates/language/URL yang sama).
2) JANGAN sesekali tulis semula pautan sebagai teks biasa (contoh, 「teks」（URL）, teks (URL)).
3) Terjemah hanya teks pautan; kekalkan struktur Markdown dan URL tanpa ubah.
4) Jangan tambah 「」 di sekitar pautan Markdown kecuali tatabahasa luar-pautan memerlukannya.

STRUKTUR LEBIH PENTING DARIPADA GAYA.
Jangan perkemas keberhasilan semula jadi Bahasa Jepun jika token Markdown akan berubah.

Contoh
Asal: Dokumen ini menggunakan [Co-op Translator](https://github.com/Azure/co-op-translator).
