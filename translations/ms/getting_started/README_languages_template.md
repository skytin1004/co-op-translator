# 🌐 Sokongan Pelbagai Bahasa (Templat)

Penjaga: Blok di bawah adalah contoh "semua bahasa" yang diuruskan oleh Co-op Translator.

- Jika anda mahu Co-op Translator mengekalkan senarai ini sentiasa dikemas kini secara automatik apabila anda menjalankan `translate` (pemilihan bahasa apa pun), kekalkan kedua-dua penanda komen seperti asal.
- Jika anda hanya mahu memaparkan subset bahasa sahaja, padam kedua-dua penanda komen tersebut dan keluarkan mana-mana bahasa yang anda tidak mahu senaraikan. Selepas memadam penanda tersebut, Co-op Translator tidak akan lagi menggantikan bahagian ini secara automatik.

- Seksyen ini kini termasuk nasihat "Lebih Suka Klon Secara Tempatan?" untuk membantu pengguna mengklon tanpa muatan terjemahan yang besar. Anda boleh peribadikan nasihat ini dengan URL repositori anda dengan menjalankan, sebagai contoh:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
