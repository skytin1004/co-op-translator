# Localizeflow for Co-op Translator

## はじめに

このリポジトリは、pull request 用の GitHub App である *Co-op Translator* の翻訳管理ソリューションである Localizeflow を含んでいます。ユーザーはこのシステムにより、翻訳のプルリクエストの作成、レビュー、マージが効率的に行えます。

## 使い方概要

1. プロジェクトで Localizeflow を有効化する。
2. 新しい翻訳プルリクエストを作成する。
3. 翻訳者とレビュアーは GitHub 上で変更をレビューし、コメントしながら改善する。
4. レビューが完了したらマージして、翻訳を本プロジェクトに反映させる。

## 重要なリンク

- [Co-op Translator](https://github.com/Azure/co-op-translator)
- [Localizeflow ドキュメント](https://localizeflow.org/docs)
- [GitHub App のセットアップ方法](https://github.com/apps/co-op-translator/installations)

## 貢献方法

Localizeflow や Co-op Translator に関して貢献したい場合は、リポジトリ内の README.md を参照し、プルリクエストを作成してください。
