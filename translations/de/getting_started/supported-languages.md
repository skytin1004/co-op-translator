# Unterstützte Sprachen

Die Tabelle unten listet die Sprachen auf, die derzeit von **Co-op Translator** unterstützt werden. Sie enthält Sprachcodes, Sprachennamen und bekannte Probleme, die mit jeder Sprache verbunden sind. Wenn Sie Unterstützung für eine neue Sprache hinzufügen möchten, fügen Sie bitte den entsprechenden Sprachcode, Namen und die geeignete Schriftart in der Datei `font_language_mappings.yml` im Verzeichnis `src/co_op_translator/fonts/` hinzu und senden Sie nach dem Testen einen Pull Request ein.
