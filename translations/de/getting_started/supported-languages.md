Die nachstehende Tabelle listet die derzeit von **Co-op Translator** unterstützten Sprachen auf. Sie enthält Sprachcodes, Sprachbezeichnungen sowie bekannte Probleme für jede Sprache. Wenn Sie Unterstützung für eine neue Sprache hinzufügen möchten, fügen Sie bitte den entsprechenden Sprachcode, Namen und die passende Schriftart in der Datei `font_language_mappings.yml` im Verzeichnis `src/co_op_translator/fonts/` hinzu und senden Sie nach dem Testen einen Pull Request ein.
