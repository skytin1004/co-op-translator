# 🌐 Mehrsprachige Unterstützung (Vorlage)

Maintainer: Der untenstehende Block ist ein „Alle Sprachen“-Beispiel, das von Co-op Translator verwaltet wird.

- Wenn Sie möchten, dass Co-op Translator diese Liste bei Ausführung von `translate` (einer beliebigen Sprachauswahl) automatisch vollständig aktuell hält, belassen Sie die beiden Kommentar-Markierungen unverändert.
- Wenn Sie nur einen Teil der Sprachen anzeigen möchten, löschen Sie die beiden Kommentar-Markierungen und entfernen Sie die Sprachen, die Sie nicht auflisten möchten. Nach Entfernen der Markierungen ersetzt Co-op Translator diesen Abschnitt nicht mehr automatisch.

- Der Abschnitt enthält jetzt einen Hinweis „Bevorzugen Sie das lokale Klonen?“, um Benutzern zu helfen, ohne die große Übersetzungspayload zu klonen. Sie können den Hinweis mit Ihrer Repository-URL personalisieren, indem Sie beispielsweise ausführen:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
