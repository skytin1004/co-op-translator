# 🌐 ဘာသာစကားများစွာ အထောက်အပံ့ (အမျိုးအစား)

ထိန်းသိမ်းကြသူများ: အောက်ပါအပိုဒ်မှာ Co-op Translator က စီမံခန့်ခွဲထားသော "ဘာသာစကားအားလုံး" ဥပမာဖြစ်သည်။

- သင်က `translate` (ဘာသာစကားရွေးချယ်မှုမဆို) ကို ပစ္စည်းစဉ်ကို အပြည့်အစုံ အလိုအလျောက် အသစ်တင်လိုလျှင် မှတ်ချက်အမှတ်နှစ်ခုကို မပြောင်းလဲထားပါ။
- သင် ဗမာသာသာရေးဖြင့်သာ ပြသလိုပါက မှတ်ချက်အမှတ်နှစ်ခုကို ဖျက်ပြီး မလိုလားသော ဘာသာစကားများကို ဖယ်ရှားနိုင်သည်။ မှတ်ချက်အမှတ်များကို ဖယ်ရှားပြီးနောက် Co-op Translator က ဤအပိုင်းကို အလိုအလျောက် မပြောင်းလဲတော့ပါ။

- ယခုအပိုင်းတွင် "တစ်ကွေ့ကို အရောက်ဒေါင်းလုပ်လုပ်ရန် ဆန္ဒရှိပါသလား?" ဆိုသော သတိပေးချက်ပါဝင်ပြီး အသုံးပြုသူများအား ဘာသာပြန်ဆိုမှုများ အကြီးစားဒေတာကို မယူဘဲ ကလုံလုပ်နိုင်ရန် ကူညီပေးသည်။ သင်၏ ဂိုဒေါင် URL ဖြင့် ကိုယ်ပိုင်ပြင်ဆင်ထားနိုင်ပြီး ဥပမာအားဖြင့်:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"` ကို အသုံးပြုပါ။
