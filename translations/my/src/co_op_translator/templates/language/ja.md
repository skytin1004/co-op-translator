# GitHub App のインストール

GitHub App をインストールするには、以下の手順に従ってください。

1. GitHub の Marketplace にアクセスします。
2. インストールしたい GitHub App を検索します。
3. 「Install」ボタンをクリックします。
4. 必要な権限を確認し、リポジトリや組織を選択します。
5. インストールを完了します。

インストール後、App の機能を利用して作業を効率化しましょう。

詳しい情報は [GitHub App の公式ドキュメント](https://docs.github.com/en/developers/apps/installing-github-apps) を参照してください。
