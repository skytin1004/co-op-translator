>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse <repo_url>.git
> cd <repo_name>
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse <repo_url>.git
> cd <repo_name>
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> ၎င်းသည် သင့်အတွက် လိုအပ်သမျှ အရာအားလုံးကို ပေးပြီး ဒေါင်းလုဒ်ပို့မန်မှ တော်တော်လေး မြန်ဆန်စေပါသည်။
