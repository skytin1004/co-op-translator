> **स्थानिक क्लोन करायचा प्राधान्य?**
>
> या रेपॉजिटरीमध्ये 50+ भाषा भाषांतरांचा समावेश आहे ज्यामुळे डाउनलोड आकार लक्षणीयरीत्या वाढतो. भाषांतरांशिवाय क्लोन करण्यासाठी, sparse checkout वापरा:
>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse <repo_url>.git
> cd <repo_name>
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse <repo_url>.git
> cd <repo_name>
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> हे तुम्हाला कोर्स पूर्ण करण्यासाठी आवश्यक असलेले सर्व काही द्रुत डाउनलोडसह देते.
