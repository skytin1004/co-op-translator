नियम (पालन करणे आवश्यक आहे):
1) Markdown लिंक्स अगदी तशा ठेवा: [text](../../../../../../src/co_op_translator/templates/language/URL) -> [अनुवादित मजकूर](../../../../../../src/co_op_translator/templates/language/त्याच URL).
2) लिंक्स कधीही साध्या मजकूरात बदलू नका (उदा., 「text」（URL）, text (URL)).
3) फक्त लिंक मजकूराचा अनुवाद करा; Markdown रचना आणि URL बदलू नका.
4) Markdown लिंकच्या आजूबाजूला 「」 लावू नका जोपर्यंत बाहेरील व्याकरण आवश्यक नसेल.
