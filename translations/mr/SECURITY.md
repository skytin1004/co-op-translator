## सुरक्षा

मायक्रोसॉफ्ट आपल्या सॉफ्टवेअर उत्पादनांची आणि सेवा सुरक्षिततेबाबत गांभीर्याने घेतो, ज्यात आमच्या GitHub संस्थांमधील सर्व स्रोत कोड संग्रहालयांचा समावेश आहे, ज्यामध्ये [Microsoft](https://github.com/Microsoft), [Azure](https://github.com/Azure), [DotNet](https://github.com/dotnet), [AspNet](https://github.com/aspnet) आणि [Xamarin](https://github.com/xamarin) यांचा समावेश आहे.

आपल्याला अशी सुरक्षा कमकुवतपणा आढळली असेल जी [Microsoft च्या सुरक्षा कमकुवतपणाची व्याख्या](https://aka.ms/security.md/definition) पूर्ण करते, तर कृपया खाली दिलेल्या वर्णनाप्रमाणे आम्हाला त्याची माहिती द्या.

## सुरक्षा समस्या नोंदविणे

**कृपया सार्वजनिक GitHub समस्यांद्वारे सुरक्षा कमकुवतपणा नोंदवू नका.**

त्याऐवजी, कृपया मायक्रोसॉफ्ट सिक्युरिटी रिस्पॉन्स सेंटर (MSRC) येथे [https://msrc.microsoft.com/create-report](https://aka.ms/security.md/msrc/create-report) या संकेतस्थळावर नोंदवा.

आपण लॉगिन न करता सादर करू इच्छित असल्यास, [secure@microsoft.com](mailto:secure@microsoft.com) या ईमेलवर संदेशन पाठवा. शक्य असल्यास, आमच्या PGP कीने आपल्या संदेशाचे एन्क्रिप्शन करा; कृपया ती [Microsoft Security Response Center PGP Key page](https://aka.ms/security.md/msrc/pgp) वरून डाउनलोड करा.

आपण 24 तासांच्या आत उत्तर प्राप्त करावे. कोणत्याही कारणास्तव उत्तर न मिळाले, तर कृपया ईमेलद्वारे आम्हाला पुन्हा संपर्क करा ज्यामुळे आपला मूळ संदेश मिळाला आहे याची खात्री होईल. अधिक माहिती येथे पाहा: [microsoft.com/msrc](https://www.microsoft.com/msrc).

कृपया खालील माहिती (जितकी शक्य असेल तितकी) समाविष्ट करा जेणेकरून आम्हाला समस्येचा प्रकार आणि व्याप चांगल्या प्रकारे समजेल:

  * समस्येचा प्रकार (उदा. बफर ओव्हरफ्लो, SQL इंजेक्शन, क्रॉस-साइट स्क्रिप्टिंग, इत्यादी)
  * समस्येच्या लक्षणांशी संबंधित स्रोत फाइल्सची पूर्ण पथ
  * प्रभावित स्रोत कोडचे स्थान (टॅग/ब्रांच/कमिट किंवा थेट URL)
  * समस्या पुनरुत्पादित करण्यासाठी कोणतीही खास संरचना आवश्यक असल्यास
  * समस्या पुनरुत्पादित करण्यासाठी पावलोपावल निर्देश
  * प्रूफ-ऑफ-कॉन्सेप्ट किंवा एक्सप्लॉइट कोड (जर शक्य असेल तर)
  * समस्येचा परिणाम, ज्यामध्ये हल्लेखोर कसा त्याचा फायदा घेऊ शकतो याचा समावेश

ही माहिती आम्हाला आपल्या रिपोर्टचे त्वरीत मूल्यांकन करण्यात मदत करेल.

जर आपण बग बाउंटीसाठी रिपोर्ट करत असाल, तर अधिक संपूर्ण रिपोर्ट्सना जास्त बाउंटी पुरस्कार मिळू शकतो. कृपया आमच्या [Microsoft Bug Bounty Program](https://aka.ms/security.md/msrc/bounty) या पृष्ठावर आमच्या सक्रिय कार्यक्रमांविषयी अधिक तपशील पहा.
