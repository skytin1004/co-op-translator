# 🌐 बहुभाषिक समर्थन (टेम्पलेट)

देखभाल करणारे: खालील ब्लॉक हा Co-op Translator द्वारे व्यवस्थापित केलेला "सर्व भाषा" चे उदाहरण आहे.

- तुम्हाला Co-op Translator कडून `translate` (कोणतीही भाषा निवड) चालविल्यावर या यादीला स्वयंचलितपणे पूर्णपणे अद्ययावत ठेवायचे असल्यास, दोन टिप्पणी चिन्हे तशीच ठेवा.
- तुम्हाला फक्त काही भाषांची यादी दाखवायची असल्यास, दोन टिप्पणी चिन्हे हटवा आणि तुम्हाला नको असलेल्या भाषांना काढा. चिन्हे काढल्यानंतर, Co-op Translator हा विभाग स्वयंचलितरित्या बदलणार नाही.

- हा विभाग आता "स्थानिक क्लोन करायचे का?" हा सल्ला देते, ज्यामुळे वापरकर्त्यांना मोठ्या भाषांतराच्या माहितीशिवाय क्लोन करण्यास मदत होते. तुम्ही तुमच्या रिपॉझिटरी URL सह हा सल्ला वैयक्तिक करू शकता, उदाहरणार्थ:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
