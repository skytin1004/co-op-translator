# সমর্থিত ভাষাসমূহ

নীচের টেবিলটিতে বর্তমানে **Co-op Translator** দ্বারা সমর্থিত ভাষাগুলি তালিকাভুক্ত করা হয়েছে। এতে ভাষার কোড, ভাষার নাম এবং প্রতিটি ভাষার সাথে সম্পর্কিত কোনও পরিচিত সমস্যা অন্তর্ভুক্ত রয়েছে। আপনি যদি কোনও নতুন ভাষা সমর্থন যোগ করতে চান, অনুগ্রহ করে সংশ্লিষ্ট ভাষার কোড, নাম এবং উপযুক্ত ফন্ট `src/co_op_translator/fonts/` অবস্থানে থাকা `font_language_mappings.yml` ফাইলে যোগ করুন এবং পরীক্ষার পর একটি pull request জমা দিন।
