# সমর্থিত ভাষাসমূহ

নিচের টেবিলে বর্তমানে **Co-op Translator** দ্বারা সমর্থিত ভাষাগুলি তালিকাভুক্ত করা হয়েছে। এতে ভাষার কোড, ভাষার নাম এবং প্রতিটি ভাষার সাথে সম্পর্কিত যেকোনো পরিচিত সমস্যা অন্তর্ভুক্ত রয়েছে। আপনি যদি একটি নতুন ভাষার জন্য সমর্থন যোগ করতে চান, অনুগ্রহ করে `src/co_op_translator/fonts/`-এ থাকা `font_language_mappings.yml` ফাইলে সংশ্লিষ্ট ভাষার কোড, নাম এবং উপযুক্ত ফন্ট যোগ করুন এবং পরীক্ষার পর একটি pull request জমা দিন।
