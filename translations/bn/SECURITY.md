## Security

Microsoft আমাদের সফটওয়্যার পণ্য এবং পরিষেবাগুলির নিরাপত্তা গুরুত্ব দিয়ে বিবেচনা করে, যার মধ্যে আমাদের GitHub সংগঠনগুলির মাধ্যমে পরিচালিত সমস্ত সোর্স কোড রেপোজিটরি অন্তর্ভুক্ত, যেগুলির মধ্যে রয়েছে [Microsoft](https://github.com/Microsoft), [Azure](https://github.com/Azure), [DotNet](https://github.com/dotnet), [AspNet](https://github.com/aspnet) এবং [Xamarin](https://github.com/xamarin)।

যদি আপনি মনে করেন যে আপনি কোনও Microsoft-মালিকানাধীন রেপোজিটরিতে একটি নিরাপত্তা দুর্বলতা পেয়েছেন যা [Microsoft এর নিরাপত্তা দুর্বলতার সংজ্ঞা](https://aka.ms/security.md/definition) পূরণ করে, তাহলে দয়া করে নিচের বর্ণনা অনুযায়ী আমাদের এটিকে রিপোর্ট করুন।

## Reporting Security Issues

**অনুগ্রহ করে পাবলিক GitHub ইস্যুর মাধ্যমে নিরাপত্তা দুর্বলতা রিপোর্ট করবেন না।**

এর পরিবর্তে, দয়া করে Microsoft Security Response Center (MSRC)-তে রিপোর্ট করুন [https://msrc.microsoft.com/create-report](https://aka.ms/security.md/msrc/create-report) এ।

আপনি যদি লগ ইন না করে জমা দিতে চান, তাহলে ইমেইল পাঠান [secure@microsoft.com](mailto:secure@microsoft.com) এ। সম্ভব হলে, আমাদের PGP কী দিয়ে আপনার বার্তা এনক্রিপ্ট করুন; দয়া করে এটি ডাউনলোড করুন [Microsoft Security Response Center PGP Key পৃষ্ঠা](https://aka.ms/security.md/msrc/pgp) থেকে।

আপনি ২৪ ঘন্টার মধ্যে একটি প্রতিক্রিয়া পাবেন। যদি কোনো কারণে না পান, তাহলে নিশ্চিত করতে ইমেইলের মাধ্যমে ফলো-আপ করুন যাতে আমরা আপনার মূল বার্তা পেয়েছি। অতিরিক্ত তথ্য পাওয়া যাবে [microsoft.com/msrc](https://www.microsoft.com/msrc) এ।

দয়া করে নিম্নলিখিত অনুরোধকৃত তথ্য অন্তর্ভুক্ত করুন (যতটা সম্ভব সরবরাহ করুন) যাতে আমরা সম্ভাব্য সমস্যার প্রকৃতি এবং পরিমাণ ভালভাবে বুঝতে পারি:

  * সমস্যার ধরন (যেমন বাফার ওভারফ্লো, SQL ইনজেকশন, ক্রস-সাইট স্ক্রিপ্টিং ইত্যাদি)
  * সমস্যার প্রকাশ ঘটিয়েছে এমন সোর্স ফাইলের সম্পূর্ণ পথ
  * প্রভাবিত সোর্স কোডের অবস্থান (ট্যাগ/ব্রাঞ্চ/কমিট বা সরাসরি URL)
  * সমস্যাটি পুনরুত্পাদন করতে প্রয়োজনীয় বিশেষ কোন কনফিগারেশন থাকলে তা
  * সমস্যাটি পুনরুত্পাদনের ধাপে ধাপে নির্দেশাবলী
  * প্রুফ-অফ-কনসেপ্ট বা এক্সপ্লয়েট কোড (যদি সম্ভব হয়)
  * সমস্যাটির প্রভাব, সহ একটি আক্রমণকারী কীভাবে সমস্যাটি শোষণ করতে পারে

এই তথ্য আমাদের আপনার রিপোর্ট দ্রুত পর্যালোচনা করতে সহায়তা করবে।

আপনি যদি বাগ বাউন্টির জন্য রিপোর্ট দিচ্ছেন, আরো সম্পূর্ণ রিপোর্ট উচ্চতর বাউন্টি পুরস্কারের সাথে অবদান রাখতে পারে। আমাদের [Microsoft Bug Bounty Program](https://aka.ms/security.md/msrc/bounty) পৃষ্ঠাটিতে আমাদের সক্রিয় প্রোগ্রামসমূহ সম্পর্কে আরও বিস্তারিত তথ্য পাওয়া যাবে।
