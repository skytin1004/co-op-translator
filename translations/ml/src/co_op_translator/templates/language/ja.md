ജാപ്പനീസ് മോഡ്: Markdown ടോകൻസ് കർശനമായി സംരക്ഷിക്കുക.

നിയമങ്ങൾ (പുറമേ സ്വീകൃതമായിത്തീരണം):
1) Markdown ലിങ്കുകൾ കൃത്യമായി സൂക്ഷിക്കുക: [text](../../../../../../src/co_op_translator/templates/language/URL) -> [പരിവർത്തിത വാചകം](../../../../../../src/co_op_translator/templates/language/അത് തന്നെ URL).
2) ലിങ്കുകൾ ഒരിക്കലും പ്ലെയ്ൻ ടെക്സ്റ്റായി എഴുതരുത് (ഉദാ., 「text」（URL）, text (URL)).
3) Markdown ഘടനയും URL-യും മാറ്റാതെ, ലിങ്കിന്റെ വാചകം മാത്രമേ വിവർത്തനം ചെയ്യേ.
4) ലിങ്ക് പുറത്ത് വ്യാകരണ കാരണങ്ങൾ ഇല്ലെങ്കിൽ Markdown ലിങ്കിന് ചുറ്റുമുള്ള 「」 ചേർക്കരുത്.

ഘടന ശൈലിയിൽ നിന്ന് പ്രമുഖമാണ്.
Markdown ടോകൻസ് മാറാതിരിക്കാൻ ജാപ്പനീസ് സ്വാഭാവികത അപ്ടിമൈസ് ചെയ്യരുത്.
