# പിന്തുണയ്ക്കുന്ന ഭാഷകൾ

താഴെപ്പറയുന്ന പട്ടിക **Co-op Translator** സപ്പോർട്ട് ചെയ്യുന്ന നിലവിലെ ഭാഷകൾ ഉൾക്കൊള്ളുന്നു. അതിൽ ഭാഷകോഡ്, ഭാഷനാമം, ഓരോ ഭാഷയോടും ബന്ധപ്പെട്ടുള്ള പ്രശ്നങ്ങൾ എന്നിവ അടങ്ങിയിരിക്കുന്നു. ഒരു പുതിയ ഭാഷയ്ക്ക് സപ്പോർട് നൽകാൻ ആഗ്രഹിക്കുന്നുവെങ്കിൽ, `src/co_op_translator/fonts/` ലുള്ള `font_language_mappings.yml` ഫയലിൽ അനുയോജ്യമായ ഭാഷാ കോഡ്‌, പേര്‌, ഫോണ്റ് എന്നിവ ചേർത്ത് ടെസ്റ്റിംഗിന്റെ ശേഷം ഒരു pull request സമർപ്പിക്കുക.
