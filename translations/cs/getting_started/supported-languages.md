# Podporované jazyky

Tabulka níže uvádí jazyky, které jsou aktuálně podporovány **Co-op Translator**. Zahrnuje jazykové kódy, názvy jazyků a případné známé problémy spojené s jednotlivými jazyky. Pokud chcete přidat podporu nového jazyka, přidejte prosím odpovídající jazykový kód, název a vhodné písmo do souboru `font_language_mappings.yml` umístěného v `src/co_op_translator/fonts/` a po otestování odešlete pull request.
