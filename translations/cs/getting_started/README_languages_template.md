# 🌐 Podpora více jazyků (Šablona)

Správci: Níže uvedený blok je příkladem „všechny jazyky“ spravovaným Co-op Translator.

- Pokud chcete, aby Co-op Translator udržoval tento seznam plně aktuální automaticky při spuštění příkazu `translate` (výběr jakéhokoli jazyka), ponechte tyto dva komentářové značky beze změny.
- Pokud chcete zobrazit pouze podmnožinu jazyků, smažte tyto dvě komentářové značky a odstraňte všechny jazyky, které nechcete zobrazit. Po odstranění značek Co-op Translator již tuto sekci automaticky nepřepíše.

- Sekce nyní obsahuje upozornění „Preferujete klonování lokálně?“ pro pomoc uživatelům s klonováním bez velké přenosové zátěže překladů. Toto upozornění můžete přizpůsobit pomocí URL vaší repozitáře spuštěním, například:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
