## Bezpečnost

Microsoft bere bezpečnost našich softwarových produktů a služeb vážně, což zahrnuje všechny repozitáře zdrojového kódu spravované prostřednictvím našich GitHub organizací, které zahrnují [Microsoft](https://github.com/Microsoft), [Azure](https://github.com/Azure), [DotNet](https://github.com/dotnet), [AspNet](https://github.com/aspnet) a [Xamarin](https://github.com/xamarin).

Pokud věříte, že jste našli bezpečnostní zranitelnost v libovolném repozitáři vlastněném Microsoftem, která splňuje [definici bezpečnostní zranitelnosti Microsoftu](https://aka.ms/security.md/definition), prosím nahlaste nám ji, jak je popsáno níže.

## Hlásení bezpečnostních problémů

**Prosím, nehlaste bezpečnostní zranitelnosti prostřednictvím veřejných GitHub issues.**

Místo toho je prosím hlaste do Microsoft Security Response Center (MSRC) na [https://msrc.microsoft.com/create-report](https://aka.ms/security.md/msrc/create-report).

Pokud dáváte přednost podání bez přihlášení, zašlete e-mail na [secure@microsoft.com](mailto:secure@microsoft.com). Pokud je to možné, zašifrujte svou zprávu naším PGP klíčem; prosím stáhněte si jej ze stránky [Microsoft Security Response Center PGP Key](https://aka.ms/security.md/msrc/pgp).

Odezvu byste měli obdržet do 24 hodin. Pokud ji z nějakého důvodu neobdržíte, prosím pokračujte v komunikaci e-mailem, aby bylo zajištěno, že jsme obdrželi vaši původní zprávu. Další informace jsou k dispozici na [microsoft.com/msrc](https://www.microsoft.com/msrc).

Prosím zahrňte požadované informace uvedené níže (co nejvíce můžete poskytnout), aby nám to pomohlo lépe pochopit povahu a rozsah možného problému:

  * Typ problému (např. přetečení vyrovnávací paměti, SQL injection, cross-site scripting apod.)
  * Kompletní cesty ke zdrojovým souborům souvisejícím s manifestací problému
  * Umístění ovlivněného zdrojového kódu (tag/branch/commit nebo přímá URL)
  * Jakákoli speciální konfigurace potřebná k reprodukci problému
  * Krok za krokem instrukce k reprodukci problému
  * Ukázkový kód nebo exploit (pokud je možné)
  * Dopad problému, včetně toho, jak by mohl útočník problém využít

Tyto informace nám pomohou rychleji zpracovat váš report.

Pokud hlásíte chybu pro bug bounty, kompletnější zprávy mohou přispět k vyšší odměně. Prosím navštivte naši stránku [Microsoft Bug Bounty Program](https://aka.ms/security.md/msrc/bounty) pro více detailů o našich aktivních programech.
