जापानी मोड: Markdown टोकनहरू कडाइका साथ कायम गर्नुहोस्।

नियमहरू (अनुगमन गर्नै पर्छ):
1) Markdown लिङ्कहरू ठीक त्यसैगरी राख्ने: [text](../../../../../../src/co_op_translator/templates/language/URL) -> [अनुवादित पाठ](../../../../../../src/co_op_translator/templates/language/उही URL)।
2) लिङ्कहरू कहिल्यै सामान्य पाठ जस्तो पुनःलेखन नगर्नुहोस् (जस्तै, 「text」（URL）, text (URL))।
3) केवल लिङ्कको पाठ अनुवाद गर्नुहोस्; Markdown संरचना र URL अपरिवर्तित राख्नुहोस्।
4) Markdown लिङ्कको वरिपरि 「」 थप्नुहुँदैन जबसम्म बाहिरी लिङ्क व्याकरणले आवश्यक नपारेको होस्।

संरचना शैलीभन्दा बढी महत्त्वपूर्ण छ।
Markdown टोकनहरू परिवर्तन हुने भए जापानी प्राकृतिकतालाई सुधार नगर्नुहोस्।
