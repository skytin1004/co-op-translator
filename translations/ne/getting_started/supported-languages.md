# समर्थित भाषा

तलको तालिका मा हाल **Co-op Translator** द्वारा समर्थित भाषा सूचीबद्ध छन्। यसमा भाषा कोड, भाषा नाम, र प्रत्येक भाषासँग सम्बन्धित कुनै ज्ञात समस्याहरू समाविष्ट छन्। यदि तपाईं नयाँ भाषा समर्थन थप्न चाहनुहुन्छ भने, कृपया `src/co_op_translator/fonts/` मा अवस्थित `font_language_mappings.yml` फाइलमा उपयुक्त भाषा कोड, नाम, र उपयुक्त फन्ट थप्नुहोस् र परीक्षण पछि pull request पठाउनुहोस्।
