> **هل تفضل الاستنساخ محلياً؟**
>
> يحتوي هذا المستودع على أكثر من 50 ترجمة لغوية مما يزيد بشكل كبير من حجم التنزيل. لاستنساخ بدون الترجمات، استخدم الاستقصاء المتفرق:
>
> **باش / ماك أو إس / لينكس:**
> ```bash
> git clone --filter=blob:none --sparse <repo_url>.git
> cd <repo_name>
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **سي إم دي (ويندوز):**
> ```cmd
> git clone --filter=blob:none --sparse <repo_url>.git
> cd <repo_name>
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> هذا يعطيك كل ما تحتاجه لإكمال الدورة بتنزيل أسرع بكثير.
