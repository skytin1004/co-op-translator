# 🌐 دعم متعدد اللغات (القالب)

المشرفون: المقطع أدناه هو مثال "لكافة اللغات" تتم إدارته بواسطة Co-op Translator.

- إذا كنت ترغب في أن يحافظ Co-op Translator على تحديث هذه القائمة بالكامل تلقائيًا عند تشغيل `translate` (أي اختيار لغة)، احتفظ بعلامتي التعليق كما هما.
- إذا كنت تريد فقط عرض مجموعة فرعية من اللغات، احذف علامتي التعليق وامسح أي لغات لا تريد سردها. بعد إزالة العلامات، لن يقوم Co-op Translator باستبدال هذا القسم تلقائيًا بعد ذلك.

- يتضمن القسم الآن تحذير "هل تفضل الاستنساخ محليًا؟" لمساعدة المستخدمين على الاستنساخ دون تحميل الترجمات الكبيرة. يمكن تخصيص هذا التحذير باستخدام عنوان URL لمستودعك عن طريق تشغيل، على سبيل المثال:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`

> ```bash
> git clone --filter=blob:none --sparse https://github.com/*****.git
> cd *****
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
