# اللغات المدعومة

الجدول أدناه يدرج اللغات المدعومة حاليًا من قبل **Co-op Translator**. يتضمن رموز اللغات، أسماء اللغات، وأي مشكلات معروفة مرتبطة بكل لغة. إذا كنت ترغب في إضافة دعم للغة جديدة، يرجى إضافة رمز اللغة المناسب، الاسم، والخط المناسب في ملف `font_language_mappings.yml` الموجود في `src/co_op_translator/fonts/` وتقديم pull request بعد الاختبار.
