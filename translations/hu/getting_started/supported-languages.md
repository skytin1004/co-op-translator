# Támogatott nyelvek

Az alábbi táblázat a jelenleg a **Co-op Translator** által támogatott nyelveket tartalmazza. Megjeleníti a nyelvkódokat, nyelvneveket, valamint az adott nyelvhez ismert esetleges problémákat. Ha új nyelv támogatását szeretné hozzáadni, kérjük, vegye fel a megfelelő nyelvkódot, nevet és az odaillő betűtípust a `font_language_mappings.yml` fájlba, amely a `src/co_op_translator/fonts/` mappában található, majd tesztelés után nyújtson be egy pull request-et.
