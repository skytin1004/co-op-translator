# ローカライズフロー翻訳ガイド

このドキュメントでは、Localizeflowを用いた翻訳作業の流れとベストプラクティスを説明します。

## はじめに

Localizeflowは、GitHubでのプロジェクト翻訳を支援するためのツールです。特に[Co-op Translator](https://github.com/Azure/co-op-translator)と連携して使用されます。

### 利用する主なツール

- [Co-op Translator](https://github.com/Azure/co-op-translator)
- GitHub App

### 使用方法

1. リポジトリでプルリクエスト（PR）を作成
2. GitHub Appが自動で翻訳タスクを割り当て
3. 割り当てられた翻訳者が翻訳を実施
4. 翻訳結果を確認しマージ

## 注意事項

[!IMPORTANT]
- 文章の一貫性を保つため、用語集を必ず参照してください。
- PRを送る際はレビューを依頼し、チーム内での確認を行いましょう。
