Le tableau ci-dessous liste les langues actuellement prises en charge par **Co-op Translator**. Il inclut les codes des langues, les noms des langues et les problèmes connus associés à chaque langue. Si vous souhaitez ajouter la prise en charge d'une nouvelle langue, veuillez ajouter le code de langue correspondant, le nom, ainsi que la police appropriée dans le fichier `font_language_mappings.yml` situé à `src/co_op_translator/fonts/` et soumettre une pull request après les tests.
