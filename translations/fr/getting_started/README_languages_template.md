# 🌐 Support multilingue (Modèle)

Mainteneurs : Le bloc ci-dessous est un exemple "toutes langues" géré par Co-op Translator.

- Si vous souhaitez que Co-op Translator maintienne cette liste automatiquement à jour à chaque exécution de `translate` (quelle que soit la sélection de langue), conservez les deux marqueurs de commentaire tels quels.
- Si vous souhaitez uniquement afficher un sous-ensemble de langues, supprimez les deux marqueurs de commentaire et retirez les langues que vous ne souhaitez pas afficher. Après avoir supprimé les marqueurs, Co-op Translator ne remplacera plus automatiquement cette section.

- La section inclut désormais un conseil "Préférez-vous cloner localement ?" pour aider les utilisateurs à cloner sans le gros volume des traductions. Vous pouvez personnaliser ce conseil avec l'URL de votre dépôt en exécutant, par exemple :
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
