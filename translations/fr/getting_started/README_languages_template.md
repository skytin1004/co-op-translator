# 🌐 Support Multilingue (Modèle)

Mainteneurs : Le bloc ci-dessous est un exemple "toutes langues" géré par Co-op Translator.

- Si vous souhaitez que Co-op Translator maintienne cette liste entièrement à jour automatiquement lorsque vous exécutez `translate` (quelque soit la langue sélectionnée), conservez les deux marqueurs de commentaire tels quels.
- Si vous souhaitez seulement afficher un sous-ensemble de langues, supprimez les deux marqueurs de commentaire et retirez les langues que vous ne souhaitez pas lister. Après suppression des marqueurs, Co-op Translator ne remplacera plus automatiquement cette section.

- La section inclut maintenant un conseil "Préférez-vous cloner localement ?" pour aider les utilisateurs à cloner sans la charge utile importante des traductions. Vous pouvez personnaliser ce conseil avec l'URL de votre dépôt en exécutant, par exemple :  
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
