# Localizeflow へようこそ！

このリポジトリは、Localizeflow を学び、あなたのプロジェクトにとっての参考実装として使うためのものです。

Localizeflow フレームワークを用いて日本語のオープンソース手順書を公開し、実践形式で Localizeflow の活用法、開発のベストプラクティス、コミュニティでの協力方法を学べるようにします。

主な目的は、このリポジトリとそのドキュメントを更新していき、Localizeflow のエコシステムを育て、ローカライズ管理プロセス全体を簡素化することです。

## プロジェクトへの参加

* オープンソース貢献者向けガイドラインは [CONTRIBUTING.md](CONTRIBUTING.md) を参照してください。  
* スタイルやその他の技術的な対応ガイドラインは [STYLE.md](STYLE.md) に記載されています。  
* フィードバックやプルリクエスト (PR) での協力を歓迎します。  

プロジェクトに参加し、[GitHub App](https://github.com/apps/localizeflow) を使った PR により効率的な翻訳レビューに参加してください。

ご質問や議論は、GitHub Discussions もご利用いただけます。

## 内容の構成

このリポジトリの構成は以下の通りです:

- `docs/` - メインの技術文書および Localizeflow に関するチュートリアルを含む。  
- `examples/` - Localizeflow の実装サンプルプロジェクト。  
- `tools/` - ローカライズやドキュメント管理のためのスクリプトやユーティリティ。  

## 推奨環境

Localizeflow とこのリポジトリの開発・翻訳作業は以下の環境を推奨します：

- Node.js 14 以上  
- 最新の web ブラウザ（Chrome, Firefox, Edge など）  
- [Visual Studio Code](https://code.visualstudio.com/) 推奨
