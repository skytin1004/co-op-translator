# 🌐 Многоязычная поддержка (шаблон)

Maintainers: Блок ниже — пример для «всех языков», управляемый Co-op Translator.

- Если вы хотите, чтобы Co-op Translator автоматически поддерживал этот список в актуальном состоянии при выполнении `translate` (для любого выбранного языка), оставьте два комментария без изменений.
- Если вы хотите показывать только часть языков, удалите два комментария и удалите языки, которые не хотите показывать. После удаления комментариев Co-op Translator перестанет автоматически заменять этот раздел.

- В разделе теперь есть рекомендация «Предпочитаете клонировать локально?», чтобы помочь пользователям клонировать без большого объема переводов. Вы можете персонализировать рекомендацию, указав URL вашего репозитория, например:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
