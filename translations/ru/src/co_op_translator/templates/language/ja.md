# 英和辞書の改善

このリポジトリはオープンソースプロジェクト「Localizeflow」の一部として開発されており、英和辞書ファイル `words.json` の改善を目的としています。  

## 目的:

- 誤訳や不自然な訳語の修正。
- 新しい単語や用語の追加。
- 既存の訳語の統一。  

## 貢献の方法:

1. GitHub上でリポジトリをフォークします。  
2. `words.json` を編集して改善を加えます。  
3. 変更内容をコミットしてプッシュします。  
4. 修正内容を説明するPR(pull request)を送ります。  

## 注意事項:

- 既存のMarkdown構造やリンクは変更しないでください。  
- 主要な用語（Localizeflow、Co-op Translatorなど）は翻訳せずにそのまま残してください。  
- PRでは、改善の理由を明確に記載してください。  

皆様のご協力をお待ちしています！
