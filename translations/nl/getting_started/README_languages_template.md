# 🌐 Meertalige Ondersteuning (Sjabloon)

Beheerders: Het onderstaande blok is een voorbeeld voor "alle talen" beheerd door Co-op Translator.

- Als je wilt dat Co-op Translator deze lijst volledig up-to-date houdt wanneer je `translate` uitvoert (elke taalkeuze), behoud dan de twee commentaarmarkeringen zoals ze zijn.
- Als je alleen een subset van talen wilt tonen, verwijder dan de twee commentaarmarkeringen en haal de talen die je niet wilt vermelden weg. Na het verwijderen van de markeringen zal Co-op Translator dit gedeelte niet langer automatisch vervangen.

- De sectie bevat nu een advies "Liever Lokaal Clonen?" om gebruikers te helpen clonen zonder de grote vertalingenpayload. Je kunt het advies personaliseren met je repository-URL door bijvoorbeeld het volgende uit te voeren:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
