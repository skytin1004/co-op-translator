# 🌐 Ondersteuning voor Meerdere Talen (Template)

Beheerders: De onderstaande blok is een voorbeeld voor "alle talen" beheerd door Co-op Translator.

- Als je wilt dat Co-op Translator deze lijst volledig automatisch bijwerkt wanneer je `translate` uitvoert (ongeacht taalkeuze), laat dan de twee commentaarkoppen onaangeroerd.
- Als je slechts een deel van de talen wilt tonen, verwijder dan de twee commentaarkoppen en verwijder elke taal die je niet wilt vermelden. Na het verwijderen van de markers zal Co-op Translator dit gedeelte niet meer automatisch vervangen.

- De sectie bevat nu een waarschuwing "Voorkeur om lokaal te klonen?" om gebruikers te helpen klonen zonder de grote vertalingspayload. Je kunt deze waarschuwing personaliseren met je repository-URL door bijvoorbeeld het volgende uit te voeren:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
