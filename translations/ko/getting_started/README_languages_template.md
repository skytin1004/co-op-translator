# 🌐 다국어 지원 (템플릿)

유지 관리자: 아래 블록은 Co-op Translator가 관리하는 "모든 언어" 예시입니다.

- Co-op Translator가 `translate`를 실행할 때(언어 선택에 관계없이) 이 목록을 자동으로 최신 상태로 유지하도록 하려면, 두 개의 주석 표시를 그대로 유지하세요.
- 일부 언어만 표시하려면 두 개의 주석 표시를 삭제하고 나열하지 않을 언어를 제거하세요. 주석 표시를 제거하면 Co-op Translator가 이 섹션을 더 이상 자동으로 대체하지 않습니다.

- 이 섹션에는 이제 대용량 번역 콘텐츠 없이도 클론할 수 있도록 안내하는 "로컬로 클론하는 것을 선호하나요?"라는 권고가 포함되어 있습니다. 예를 들어 다음 명령어를 실행하여 저장소 URL로 권고 문구를 개인화할 수 있습니다:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
