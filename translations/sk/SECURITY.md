## Security

Microsoft berie bezpečnosť našich softvérových produktov a služieb vážne, čo zahŕňa všetky repozitáre so zdrojovým kódom spravované prostredníctvom našich GitHub organizácií, ktoré zahŕňajú [Microsoft](https://github.com/Microsoft), [Azure](https://github.com/Azure), [DotNet](https://github.com/dotnet), [AspNet](https://github.com/aspnet) a [Xamarin](https://github.com/xamarin).

Ak veríte, že ste našli bezpečnostnú zraniteľnosť v akomkoľvek repozitári vlastnenom spoločnosťou Microsoft, ktorá spĺňa [definíciu bezpečnostnej zraniteľnosti spoločnosti Microsoft](https://aka.ms/security.md/definition), nahláste nám ju, ako je uvedené nižšie.

## Nahlasovanie bezpečnostných problémov

**Prosíme, nenahlasujte bezpečnostné zraniteľnosti prostredníctvom verejných GitHub issues.**

Namiesto toho ich prosím nahláste na Microsoft Security Response Center (MSRC) na [https://msrc.microsoft.com/create-report](https://aka.ms/security.md/msrc/create-report).

Ak uprednostňujete odoslanie bez prihlásenia, pošlite e-mail na [secure@microsoft.com](mailto:secure@microsoft.com). Ak je to možné, zašifrujte svoju správu naším PGP kľúčom; stiahnite si ho prosím z [Microsoft Security Response Center PGP Key page](https://aka.ms/security.md/msrc/pgp).

Očakávajte odpoveď do 24 hodín. Ak z nejakého dôvodu nedostanete odpoveď, kontaktujte nás prosím znova e-mailom, aby sme sa uistili, že sme vašu pôvodnú správu prijali. Ďalšie informácie nájdete na [microsoft.com/msrc](https://www.microsoft.com/msrc).

Prosíme, zahrňte požadované informácie uvedené nižšie (čo najviac môžete poskytnúť), aby sme lepšie pochopili povahu a rozsah možného problému:

  * Typ problému (napr. pretečenie vyrovnávacej pamäte, SQL injekcia, cross-site scripting, atď.)
  * Plné cesty k súborom so zdrojovým kódom súvisiacim s prejavom problému
  * Miesto postihnutého zdrojového kódu (tag/branch/commit alebo priamy URL)
  * Akákoľvek špeciálna konfigurácia potrebná na reprodukciu problému
  * Krok za krokom inštrukcie na reprodukciu problému
  * Dôkaz konceptu alebo exploit kód (ak je možné)
  * Dopad problému, vrátane toho, ako by útočník mohol problém zneužiť

Tieto informácie nám pomôžu rýchlejšie spracovať vašu správu.

Ak nahlasujete chybu v rámci programu bug bounty, kompletné správy môžu prispieť k vyššej odmeny. Navštívte prosím našu stránku [Microsoft Bug Bounty Program](https://aka.ms/security.md/msrc/bounty) pre viac informácií o našich aktívnych programoch.
