# 🌐 Podpora viacerých jazykov (šablóna)

Správcovia: Nižšie uvedený blok je príklad „všetkých jazykov“ spravovaný pomocou Co-op Translator.

- Ak chcete, aby Co-op Translator automaticky udržiaval tento zoznam úplne aktuálny vždy, keď spustíte `translate` (výber akéhokoľvek jazyka), nechajte dva komentáre neporušené.
- Ak chcete zobraziť iba podmnožinu jazykov, odstráňte dva komentáre a zmažte jazyky, ktoré nechcete zobrazovať. Po odstránení týchto komentárov už Co-op Translator túto sekciu automaticky nenahradí.

- Sekcia teraz obsahuje upozornenie „Uprednostňujete klonovanie lokálne?“, ktoré pomáha používateľom klonovať bez veľkej záťaže prekladov. Môžete si prispôsobiť toto upozornenie pomocou URL vášho repozitára napríklad spustením:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
