# ಬೆಂಬಲಿತ ಭಾಷೆಗಳು

ಕೆಳಗಿನ ಟೇಬಲ್‌ನಲ್ಲಿ **Co-op Translator** ಪ್ರಸ್ತುತ ಬೆಂಬಲಿಸುವ ಭಾಷೆಗಳಾದ ಭಾಷಾ ಕೋಡ್ಗಳು, ಭಾಷಾ ಹೆಸರುಗಳು ಮತ್ತು ಪ್ರತಿ ಭಾಷೆಗೆ ಸಂಬಂಧಪಟ್ಟ ಯಾವುದೇ ಪ್ರಸಿದ್ಧ ಸಮಸ್ಯೆಗಳಿವೆ. ನೀವು ಹೊಸ ಭಾಷೆಗೆ ಬೆಂಬಲವನ್ನು ಸೇರಿಸಲು ಇಚ್ಛಿಸುವಿದ್ದರೆ, ದಯವಿಟ್ಟು ಹೊಂದಾಣಿಕೆಯ ಭಾಷಾ ಕೋಡ್, ಹೆಸರು ಮತ್ತು ಸರಿಯಾದ ಫಾಂಟ್ ಅನ್ನು `src/co_op_translator/fonts/` ನಲ್ಲಿ ಇರುವ `font_language_mappings.yml` ಫೈಲ್‌ಗೆ ಸೇರಿಸಿ, ಪರೀಕ್ಷೆಯಾದ ನಂತರ ಒಂದು pull request ಸಲ್ಲಿಸಿ.
