# ಬೆಂಬಲಿತ ಭಾಷೆಗಳು

ಕೆಳಗಿನ ಪಟ್ಟಿಯಲ್ಲಿ **Co-op Translator** ಮೂಲಕ ಪ್ರಸ್ತುತ ಬೆಂಬಲಿತ ಭಾಷೆಗಳ ವಿವರ ಇದೆ. ಇದರಲ್ಲಿ ಭಾಷೆ ಕೋಡ್‌ಗಳು, ಭಾಷೆಗಳ ಹೆಸರುಗಳು ಮತ್ತು ಪ್ರತಿಯೊಬ್ಬ ಭಾಷೆಗೆ ಸಂಬಂಧಿಸಿದ ಯಾವುದೇ ತಿಳಿದಿರುವ ಸಮಸ್ಯೆಗಳಿವೆ. ನೀವು ಹೊಸ ಭಾಷೆವನ್ನು ಸೇರಿಸಲು ಆಸಕ್ತರಾಗಿದ್ದರೆ, ದಯವಿಟ್ಟು `src/co_op_translator/fonts/` ನಲ್ಲಿ ಇರುವ `font_language_mappings.yml` ಫೈಲ್‌ನಲ್ಲಿ ಸಂಬಂಧಿತ ಭಾಷಾ ಕೋಡ್, ಹೆಸರು ಮತ್ತು ಸೂಕ್ತ ಫಾಂಟ್ ಅನ್ನು ಸೇರಿಸಿ, ಮತ್ತು ಪರೀಕ್ಷಿಸಿಕೊಂಡ ನಂತರ pull request ಸಲ್ಲಿಸಿ.
