## ಭದ್ರತೆ

ನಮ್ಮ GitHub ಸಂಸ್ಥೆಗಳ ಮೂಲಕ ನಿರ್ವಹಿಸಲಾಗುವ ಎಲ್ಲಾ ಮೂಲ ಕೋಡ್ ಸಂಗ್ರಹಣೆಗಳನ್ನು ಒಳಗೊಂಡು, ನಮ್ಮ ಸಾಫ್ಟ್‌ವೇರ್ ಉತ್ಪನ್ನಗಳು ಮತ್ತು ಸೇವೆಗಳ ಭದ್ರತೆಯನ್ನು ಮೈಕ್ರೋಸಾಫ್ಟ್ ಗಂಭೀರವಾಗಿ ತೆಗೆದುಕೊಳ್ಳುತ್ತದೆ. ಇದರಲ್ಲಿವೆ [Microsoft](https://github.com/Microsoft), [Azure](https://github.com/Azure), [DotNet](https://github.com/dotnet), [AspNet](https://github.com/aspnet) ಮತ್ತು [Xamarin](https://github.com/xamarin).

ನೀವು ಯಾವುದೇ Microsoft ಹೊಂದಿಸುವ ಸಂಗ್ರಹಣೆಯಲ್ಲಿ [Microsoftನ ಭದ್ರತಾ ದುರ್ಬಲತೆ ವ್ಯಾಖ್ಯಾನ](https://aka.ms/security.md/definition) ಹೊಂದಿರುವ ಭದ್ರತಾ ದುರ್ಬಲತೆಯನ್ನು ಕಂಡುಹಿಡಿದಿದ್ದೀರಿ ಎಂದು ಭಾವಿಸಿದರೆ, ದಯವಿಟ್ಟು ಕೆಳಗಿನಂತೆ ನಮಗೆ ವರದಿ ಮಾಡಿ.

## ಭದ್ರತಾ ಸಮಸ್ಯೆಗಳ ವರದಿ

**ದಯವಿಟ್ಟು ಭದ್ರತಾ ದುರ್ಬಲತೆಗಳನ್ನು ಸಾರ್ವಜನಿಕ GitHub ಸಮಸ್ಯೆಗಳ ಮೂಲಕ ವರದಿ ಮಾಡಬೇಡಿ.**

ಬದಲಿಗೆ, ದಯವಿಟ್ಟು Microsoft Security Response Center (MSRC) ನಲ್ಲಿ [https://msrc.microsoft.com/create-report](https://aka.ms/security.md/msrc/create-report) ಮೂಲಕ ವರದಿ ಮಾಡಿರಿ.

ಲೇಖನ ಲಾಗಿನ್ ಮಾಡದೇ ಸಲ್ಲಿಸಲು ಬಯಸಿದರೆ, [secure@microsoft.com](mailto:secure@microsoft.com) ಗೆ ಇಮೇಲ್ ಕಳುಹಿಸಿ. ಸಾಧ್ಯವಾದರೆ, ನಮ್ಮ PGP ಕೀ ಜೊತೆಗೆ ನಿಮ್ಮ ಸಂದೇಶವನ್ನು ಎನ್‌ಕ್ರಿಪ್ಟ್ ಮಾಡಿ; ದಯವಿಟ್ಟು ಅದನ್ನು [Microsoft Security Response Center PGP Key ಪುಟದಿಂದ](https://aka.ms/security.md/msrc/pgp) ಡೌನ್‌ಲೋಡ್ ಮಾಡಿ.

ನೀವು 24 ಗಂಟೆಗಳಲ್ಲಿ ಪ್ರತಿಕ್ರಿಯೆ ಪಡೆಯಬೇಕು. ಯಾವುದೇ ಕಾರಣಕ್ಕಾಗಿ ಇದಾಗದಿದ್ದರೆ, ದಯವಿಟ್ಟು ನಿಮ್ಮ ಮೂಲ ಸಂದೇಶವನ್ನು ನಾವು ಪಡೆದಿರುವುದನ್ನು ಖಚಿತಪಡಿಸಲು ಇಮೇಲ್ ಮೂಲಕ ಹಿಂಬಾಲಿಸಿ. ಹೆಚ್ಚಿನ ಮಾಹಿತಿಯನ್ನು [microsoft.com/msrc](https://www.microsoft.com/msrc) ನಲ್ಲಿ ಕಾಣಬಹುದು.

ಕೇಳಲಾದ ಕೆಳಗಿನ ಮಾಹಿತಿಯನ್ನು (ನೀವು ನೀಡಬಹುದಾದಷ್ಟು) ಸೇರಿಸಿ, ಸಮಸ್ಯೆಯ ಸ್ವಭಾವ ಮತ್ತು ವ್ಯಾಪ್ತಿಯನ್ನು ನಮಗೆ ಉತ್ತಮವಾಗಿ ಅರ್ಥಮಾಡಿಕೊಳ್ಳಲು ಸಹಾಯ ಮಾಡಿ:

  * ಸಮಸ್ಯೆಯ ಪ್ರಕಾರ (ಉದಾ: ಬಫರ್ ಓವರ್‌ಫ್ಲೋ, SQL ಇಂಜಕ್ಷನ್, ಕ್ರಾಸ್-ಸೈಟ್ ಸ್ಕ್ರಿಪ್ಟಿಂಗ್, ಮುಂತಾದವು)
  * ಸಮಸ್ಯೆಯ ಪ್ರकटವಾಗುವ ಸಂಬಂಧಿಸಿದ ಮೂಲ ფೈಲ್‌ಗಳ ಸಂಪೂರ್ಣ ಮಾರ್ಗಗಳು
  * ಪರಿಣಾಮಗೊಳ್ಳುವ ಮೂಲ ಕೋಡ್‌ನ ಸ್ಥಳ (ಟ್ಯಾಗ್/ಶಾಖೆ/ಕಮಿಟ್ ಅಥವಾ ನೇರ URL)
  * ಸಮಸ್ಯೆಯನ್ನು ಪುನರುತ್ಪಾದಿಸಲು ಅಗತ್ಯವಿರುವ ವಿಶೇಷ ಸಂರಚನೆ
  * ಸಮಸ್ಯೆಯನ್ನು ಪುನರುತ್ಪಾದಿಸುವ ಹಂತ-ಹಂತದ ಸೂಚನೆಗಳು
  * ನಿರೂಪಣೆ-ಆಧಾರ ಅಥವಾ ದುರ್ಬಲತೆ ಸಂಹಿತೆ (ಸಾಧ್ಯವಾದರೆ)
  * ಸಮಸ್ಯೆಯ ಪ್ರಭಾವ, ಮತ್ತು ದಾಳಿದಾರರು ಅದನ್ನು ಹೇಗೆ ದುರುಪಯೋಗ ಮಾಡಬಹುದು

ಈ ಮಾಹಿತಿ ನಮಗೆ ನಿಮ್ಮ ವರದಿಯನ್ನು ವೇಗವಾಗಿ ತಿರುಗಿಸಿಡಲು ಸಹಾಯ ಮಾಡುತ್ತದೆ.

ನೀವು ಬಗ್ಗಿ ಬಿಡಾವಣೆಯಾಗಿ ವರದಿ ಮಾಡುತ್ತಿದ್ದರೆ, ಸಂಪೂರ್ಣ ವರದಿಗಳು ಹೆಚ್ಚಿನ ಬಿಡಾವಣೆ ಬಹುಮಾನಕ್ಕೆ ಸಹಾಯ ಮಾಡಬಹುದು. ದಯವಿಟ್ಟು ನಮ್ಮ [Microsoft Bug Bounty Program](https://aka.ms/security.md/msrc/bounty) ಪುಟವನ್ನು ಕಾರ್ಯಕ್ರಮಗಳ ಬಗ್ಗೆ ಹೆಚ್ಚಿನ ವಿವರಗಳಿಗೆ ಭೇಟಿ ನೀಡಿ.

## ಆದ್ಯತೆಯ ಭಾಷೆಗಳು

ನಾವು ಎಲ್ಲಾ ಸಂವಹನಗಳು ಇಂಗ್ಲಿಷ್‌ನಲ್ಲಿ ಇರಬೇಕೆಂದು ಆಶಿಸುತ್ತೇವೆ.
