# 🌐 Suporta sa Maraming Wika (Template)

Mga Tagapangasiwa: Ang block sa ibaba ay isang halimbawa para sa "lahat ng wika" na pinamamahalaan ng Co-op Translator.

- Kung nais mong panatilihing ganap na napapanahon ng Co-op Translator ang listahang ito nang awtomatiko kapag nagpatakbo ka ng `translate` (anumang pagpili ng wika), panatilihin ang dalawang comment markers na wala ding pagbabago.
- Kung nais mo lamang ipakita ang isang bahagi ng mga wika, tanggalin ang dalawang comment markers at alisin ang anumang mga wikang ayaw mong isama. Pagkatapos tanggalin ang mga marker, hindi na aayusin nang awtomatiko ng Co-op Translator ang seksyong ito.

- Kasama na ngayon sa seksyon ang isang "Mas gusto bang Mag-clone nang Lokal?" na paalala upang tulungan ang mga gumagamit na mag-clone nang hindi kasama ang malalaking payload ng pagsasalin. Maaari mong personalisin ang paalala gamit ang iyong URL ng repositoryo sa pamamagitan ng pagpapatakbo, halimbawa:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
