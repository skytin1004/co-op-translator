# 🌐 Suporta sa Maramihang Wika (Template)

Mga Tagapanatili: Ang bloke sa ibaba ay isang halimbawa para sa "lahat ng wika" na pinamamahalaan ng Co-op Translator.

- Kung gusto mong panatilihing ganap na napapanahon ng Co-op Translator ang listahang ito nang awtomatiko kapag nagpapatakbo ka ng `translate` (anuman ang pagpili ng wika), panatilihin ang dalawang marker ng komento nang buo.
- Kung nais mo lamang ipakita ang isang bahagi ng mga wika, tanggalin ang dalawang marker ng komento at alisin ang anumang mga wika na ayaw mong ipakita. Pagkatapos tanggalin ang mga marker, hindi na awtomatikong papalitan ng Co-op Translator ang seksyong ito.

- Kasama na ngayon sa seksyon ang isang paalala na "Mas Gustong I-clone Nang Lokal?" upang tulungan ang mga user na mag-clone nang hindi kasama ang malalaking payload ng pagsasalin. Maaari mong i-personalize ang paalala gamit ang URL ng iyong repositoryo sa pamamagitan ng pagpapatakbo, halimbawa:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
