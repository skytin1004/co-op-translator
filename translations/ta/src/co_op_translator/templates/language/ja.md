ஜப்பானிய முறையில்: Markdown குறிச்சொற்களை கடுமையான முறையில் பாதுகாக்கவும்.

விதிகள் (அனுசரிக்க வேண்டும்):
1) Markdown பிணைப்புகளை ஒரேபோல் வைத்துக் கொள்ளவும்: [text](../../../../../../src/co_op_translator/templates/language/URL) -> [மொழிமாற்றப்பட்ட உரை](../../../../../../src/co_op_translator/templates/language/அதே URL).
2) பிணைப்புகளை எப்போதும் ஒற்றை உரையாக மறுபிரதி செய்யாதீர்கள் (எ.கா., 「text」（URL）, text (URL)).
3) Markdown அமைப்பையும் URL-ஐ மாற்றாமலேயே, பிணைப்புக் உரையை மட்டுமே மொழிமாற்றம் செய்யவும்.
4) பிணைப்பு அட்டவணை புறத்திற்கு வெளியே உள்ள பண்பாட்டுக் குறியீடு தேவையென்றால் தவிர, Markdown பிணைப்புக்கு 「」 சேர்க்காதீர்கள்.

அமைப்பு முறையே பாணிக்கு மேலானது.
Markdown குறிச்சொற்கள் மாறுவதைத் தவிர எந்த அளவிலும் ஜப்பானிய இயல்புத்தன்மையை மேம்படுத்த வேண்டாம்.
