> **உள்ளூர் குளோன் செய்ய விரும்புகிறீர்களா?**
>
> இந்த சேமிப்பகம் 50+ மொழி மொழிபெயர்ப்புகளை உள்ளடக்கியுள்ளது, இது загрузка அளவை பெரிதாக அதிகரிக்கிறது. மொழிபெயர்ப்புகள் இல்லாமல் குளோன் செய்ய sparse checkout ஐ பயன்படுத்தவும்:
>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse <repo_url>.git
> cd <repo_name>
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse <repo_url>.git
> cd <repo_name>
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> இதன் மூலம் படிப்பை முடிக்க தேவையான அனைத்தும் மிக வேகமாக பதிவிறக்கம் செய்யப்படும்.
