# 🌐 பல மொழி ஆதரவு (தரவு மாதிரி)

பராமரிப்பவர்கள்: கீழ்காணும் பிரிவு Co-op Translator மூலம் நிர்வகிக்கப்படும் "அனைத்து மொழிகள்" உதாரணமாகும்.

- நீங்கள் `translate` (ஏதாவது மொழி தெரிவு) இயங்கும்போது Co-op Translator இந்த பட்டியலை முழுமையாக தானாக புதுப்பிக்க வைக்க விரும்பினால், இரண்டு கருத்து குறிமுகட்டுகளை அப்படியே வைக்கவும்.
- நீங்கள் மொழிகளில் சிலவற்றை மட்டும் காட்ட விரும்பினால், இரு கருத்து குறிமுகட்டுகளையும் நீக்கி நீங்கள் காண்பிக்க விரும்பாத மொழிகளை நீக்கவும். குறிமுகட்டுகள் அகற்றப்பட்ட பிறகு, Co-op Translator இந்த பிரிவை தானாக மாற்றாது.

- இப்போது இந்தப் பகுதியிலுள்ள ஒரு "உடனடி உள்ளூரில் கிளோன் பண்ண விரும்புகிறீர்களா?" எனும் அறிவிப்பு பெரிய மொழிபெயர்ப்பு தரவை இழக்காம கிளோன் பண்ண உதவுகிறது. நீங்கள் உங்கள் ரெப்போசிடரி URL-ஐ கொண்டு அறிவிப்பை தனிப்பயனாக்க முடியும், உதாரணமாக:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
