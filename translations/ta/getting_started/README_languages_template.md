# 🌐 பன்மொழி ஆதரவு (வெர்சன்)

பராமரிப்பவர்கள்: கீழ்க்காணும் பிளாக் Co-op Translator மூலம் நிர்வகிக்கப்படும் "எல்லா மொழிகளும்" எடுத்துக்காட்டு ஆகும்.

- நீங்கள் `translate` (ஏதாவது மொழி தேர்வு) இயக்கும் போது Co-op Translator இந்த பட்டியலை முழுமையாக தானாகவே புதுப்பிக்க விரும்பினால், இரு கருத்து குறிகள் அப்படியே இருக்க வேண்டும்.
- நீங்கள் ஒரு பகுதியான மட்டும் மொழிகளை காட்ட விரும்பினால், இரு கருத்து குறிகளை நீக்கி, பட்டியலிலிருந்து தேவையில்லாத மொழிகளை நீக்கவும். குறிகளை நீக்கிய பிறகு, Co-op Translator இந்த பகுதியை தானாக மாற்றாது.

- இப்பகுதியில் இப்போது "புதிய பிரதி உள்ளூர் நிலையில் உருவாக்க விரும்புகிறீர்களா?" என்ற அறிவுறுத்தலும் உள்ளடக்கப்பட்டுள்ளது, இது பெரிய மொழிபெயர்ப்பு பதிவிறக்கம் இல்லாமல் கிளோன் செய்ய பயனர்களுக்கு உதவும். உதாரணமாக, உங்கள் சேமிப்பக URL-ஐ தனிப்பயனாக்க இந்தக் கட்டளையை இயக்கலாம்:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
