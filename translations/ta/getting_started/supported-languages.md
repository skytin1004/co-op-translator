# ஆதரவு அளிக்கப்பட்ட மொழிகள்

கீழே உள்ள அட்டவணையில் **Co-op Translator** தற்போதைய ஆதரவு அளிக்கும் மொழிகளின் பட்டியல் உள்ளது. இதில் மொழி குறியீடுகள், மொழி பெயர்கள் மற்றும் ஒவ்வொரு மொழிக்கும் தொடர்புடைய அறியப்பட்ட பிரச்சினைகள் உள்ளன. நீங்கள் புதிய ஒரு மொழிக்கான ஆதரவை சேர்க்க விரும்பினால், தயவுசெய்து `src/co_op_translator/fonts/` இல் உள்ள `font_language_mappings.yml` கோப்பில் தொடர்புடைய மொழி குறியீடு, பெயர் மற்றும் சரியான எழுத்துருவை சேர்க்கவும் மற்றும் சோதனை செய்த பிறகு pull request சமர்ப்பிக்கவும்.
