# ஆதரிக்கப்படும் மொழிகள்

கீழ்காணும் அட்டவணையில் **Co-op Translator** இல் தற்போது ஆதரிக்கப்படும் மொழிகள் பட்டியலிடப்பட்டுள்ளன. இதில் மொழி குறியீடுகள், மொழி பெயர்கள் மற்றும் ஒவ்வொரு மொழியுடனும் தொடர்புடைய எதாவது பிரச்சனைகள் உள்ளன. புதிய மொழி ஒன்றைச் சேர்க்க விரும்பினால், தயவுசெய்து `src/co_op_translator/fonts/` இல் உள்ள `font_language_mappings.yml` கோப்பில் தொடர்புடைய மொழி குறியீடு, பெயர் மற்றும் பொருத்தமான எழுத்துருவை சேர்க்கவும், உள்ளடக்கம் சோதித்த பிறகு pull request சமர்ப்பிக்கவும்.
