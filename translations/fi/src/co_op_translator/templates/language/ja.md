# Flutter このウィジェットカタログについて

このリポジトリはMaterial DesignとCupertinoのFlutterウィジェットカタログです。  

- Androidの人は、[起動はこちら](https://play.google.com/store/apps/details?id=com.flutter.catalog)をクリックしてください。  
- iOSの人は、[起動はこちら](https://apps.apple.com/jp/app/flutter-widgets-catalog/id1438055232)をクリックしてください。  

## コンテンツ

ウィジェットの一覧がとても多いです。  
以下のように分類されています。

- [Material](https://flutter.dev/docs/development/ui/widgets/material) Widget  
- [Cupertino](https://flutter.dev/docs/development/ui/widgets/cupertino) Widget  
- Layout Widget  
- Text Widget  
- Advanced Widget  
- Other Widget  

ウィジェットは日本語で説明されています。

## 開発について

このアプリは[Flutter](https://flutter.dev/)で書かれています。  
このアプリ開発のサンプルとしてご利用ください。  

また、GitHubのこのリポジトリは「Localizeflow」で翻訳されています。  
 Localizationに関わる人は[Localizeflow公式サイト](https://localizeflow.com/)をご覧ください。

## その他

- Flutter 2.5.0以降を推奨しています。  
- [GitHub](https://github.com/flutter/flutter)を参照。
- 使い方に関する質問は[Flutter公式フォーラム](https://flutter.dev/community)をご利用ください。
