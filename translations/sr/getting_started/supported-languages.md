# Подржани језици

Табела испод садржи језике које тренутно подржава **Co-op Translator**. Укључује кодове језика, имена језика и све познате проблеме повезане са сваким језиком. Уколико желите да додате подршку за нови језик, молимо да додате одговарајући код језика, име и одговарајући фонт у фајл `font_language_mappings.yml` који се налази у `src/co_op_translator/fonts/` и пошаљете pull request након тестирања.
