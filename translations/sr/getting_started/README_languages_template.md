# 🌐 Подршка за више језика (Шаблон)

Одржавачи: Блок испод је пример за "све језике" којим управља Co-op Translator.

- Ако желите да Co-op Translator аутоматски држи овај списак потпуно ажурним када покренете `translate` (избор било ког језика), оставите два коментара непромењена.
- Ако желите да прикажете само подскуп језика, избришите та два коментара и уклоните све језике које не желите да наведете. Након уклањања коментара, Co-op Translator више неће аутоматски заменити овај одељак.

- Одељак сада укључује упутство "Желите ли да клонирате локално?" како би се корисницима помогло да клонирају без великог преводног садржаја. Можете персонализовати упутство са URL-ом вашег репозиторијума тако што ћете покренути, на пример:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
