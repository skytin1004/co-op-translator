> **Föredrar du att Klona Lokalt?**
>
> Detta arkiv inkluderar översättningar på över 50 språk vilket avsevärt ökar nedladdningsstorleken. För att klona utan översättningar, använd sparsamt uthämtning:
>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse <repo_url>.git
> cd <repo_name>
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse <repo_url>.git
> cd <repo_name>
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> Detta ger dig allt du behöver för att genomföra kursen med en mycket snabbare nedladdning.
