# ローカライズフロー

[Localizeflow](https://localizeflow.com) は、技術文書や Web ページの国際化と翻訳を効率的に管理するためのモダンなプラットフォームです。これは、GitHub リポジトリと連携して、CI/CD パイプラインと翻訳の自動化を可能にします。

## 特徴

- GitHub App インテグレーションによる直接的な翻訳編集
- 翻訳ワークフローの自動化
- 翻訳の品質管理とレビュー機能
- 多言語の管理と一元化

## 使い方

1. GitHub リポジトリに Localizeflow GitHub App をインストールします。
2. 翻訳したい Markdown や JSON ファイルをプッシュします。
3. ローカライズフローのダッシュボードで翻訳プロジェクトを作成します。
4. Co-op Translator などのツールを使用して翻訳を開始します。
5. 翻訳が完了したら、PR を発行してレビューし、マージします。

詳細は [Localizeflow documentation](https://localizeflow.com/docs) をご覧ください。
