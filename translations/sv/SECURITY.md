## Security

Microsoft tar säkerheten för våra programvaruprodukter och tjänster på allvar, vilket inkluderar alla källkodsförråd som hanteras genom våra GitHub-organisationer, som inkluderar [Microsoft](https://github.com/Microsoft), [Azure](https://github.com/Azure), [DotNet](https://github.com/dotnet), [AspNet](https://github.com/aspnet) och [Xamarin](https://github.com/xamarin).

Om du tror att du har hittat en säkerhetssårbarhet i något Microsoft-ägt förråd som uppfyller [Microsofts definition av en säkerhetssårbarhet](https://aka.ms/security.md/definition), var god rapportera den till oss enligt beskrivningen nedan.

## Reporting Security Issues

**Rapportera inte säkerhetssårbarheter genom offentliga GitHub-ärenden.**

Rapportera dem istället till Microsoft Security Response Center (MSRC) på [https://msrc.microsoft.com/create-report](https://aka.ms/security.md/msrc/create-report).

Om du föredrar att skicka utan inloggning, skicka e-post till [secure@microsoft.com](mailto:secure@microsoft.com). Om möjligt, kryptera ditt meddelande med vår PGP-nyckel; vänligen ladda ner den från sidan [Microsoft Security Response Center PGP Key](https://aka.ms/security.md/msrc/pgp).

Du bör få ett svar inom 24 timmar. Om du av någon anledning inte gör det, följ upp via e-post för att säkerställa att vi mottagit ditt ursprungliga meddelande. Ytterligare information finns på [microsoft.com/msrc](https://www.microsoft.com/msrc).

Vänligen inkludera den begärda informationen nedan (så mycket du kan förse) för att hjälpa oss att bättre förstå arten och omfattningen av det möjliga problemet:

  * Typ av problem (t.ex. buffertöverskridning, SQL-injektion, cross-site scripting, etc.)
  * Fullständiga sökvägar till källfil(er) relaterade till problemet
  * Platsen för den berörda källkoden (tagg/gren/commit eller direkt URL)
  * Eventuell speciell konfiguration som krävs för att återskapa problemet
  * Steg-för-steg instruktioner för att återskapa problemet
  * Proof-of-concept eller exploateringskod (om möjligt)
  * Problemet påverkan, inklusive hur en angripare kan utnyttja problemet

Denna information hjälper oss att snabbare hantera din rapport.

Om du rapporterar för en bug bounty, kan mer kompletta rapporter bidra till en högre belöning. Besök gärna vår [Microsoft Bug Bounty Program](https://aka.ms/security.md/msrc/bounty)-sida för mer information om våra aktiva program.
