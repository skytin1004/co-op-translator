# 🌐 Fler språkstöd (Mall)

Underhållare: Blocket nedan är ett exempel för "alla språk" som hanteras av Co-op Translator.

- Om du vill att Co-op Translator ska hålla denna lista helt uppdaterad automatiskt när du kör `translate` (val av språk), behåll de två kommentarmarkörerna som de är.
- Om du bara vill visa en delmängd av språk, ta bort de två kommentarmarkörerna och ta bort de språk du inte vill visa. Efter att ha tagit bort markörerna kommer Co-op Translator inte längre att automatiskt ersätta denna sektion.

- Sektionen innehåller nu en "Föredrar du att klona lokalt?"-anvisning för att hjälpa användare att klona utan den stora översättningsmängden. Du kan anpassa anvisningen med din repository-URL genom att köra till exempel:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
