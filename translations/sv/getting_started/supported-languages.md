# Stödda språk

Tabellen nedan listar de språk som för närvarande stöds av **Co-op Translator**. Den inkluderar språkkoder, språknamn och eventuella kända problem kopplade till varje språk. Om du vill lägga till stöd för ett nytt språk, vänligen lägg till motsvarande språkkod, namn och lämpligt typsnitt i filen `font_language_mappings.yml` som finns i `src/co_op_translator/fonts/` och skicka in en pull request efter testning.
