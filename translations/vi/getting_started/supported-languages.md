# Các ngôn ngữ được hỗ trợ

Bảng dưới đây liệt kê các ngôn ngữ hiện đang được **Co-op Translator** hỗ trợ. Nó bao gồm mã ngôn ngữ, tên ngôn ngữ và bất kỳ vấn đề đã biết nào liên quan đến từng ngôn ngữ. Nếu bạn muốn thêm hỗ trợ cho một ngôn ngữ mới, vui lòng thêm mã ngôn ngữ tương ứng, tên và phông chữ phù hợp vào tệp `font_language_mappings.yml` nằm trong `src/co_op_translator/fonts/` và gửi pull request sau khi đã thử nghiệm.
