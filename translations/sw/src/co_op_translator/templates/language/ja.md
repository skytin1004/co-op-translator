---
title: "使い始め"
description: "Localizeflow を使って最初の翻訳リクエストを作成する方法の説明"
---

Localizeflow では、[Co-op Translator](https://github.com/Azure/co-op-translator) リポジトリの上に基づき作られた GitHub App と pull request (PR) を使って複数の人が一緒に翻訳できます。

- [Co-op Translator](https://github.com/Azure/co-op-translator) GitHub アカウントをお持ちでない場合、まずは登録してください。

- 翻訳者としてプロジェクトに追加されるのを待ってください。

- 追加されたら、PR で「翻訳したい」旨をコメントして始められます。

人数制限はありません。最初の翻訳者として始めるには、PR の説明文で役割を宣言してください。

(詳細は [Co-op Translator ドキュメンテーション](https://github.com/Azure/co-op-translator/blob/main/docs/README.md)をご参照ください。)
