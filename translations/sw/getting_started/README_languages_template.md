# 🌐 Msaada wa Lugha Nyingi (Templet)

Watunzaji: Sehemu iliyo hapa chini ni mfano wa "lugha zote" unaosimamiwa na Co-op Translator.

- Ikiwa unataka Co-op Translator idumishe orodha hii kuwa ya kisasa moja kwa moja unapotekeleza `translate` (chaguzi yoyote ya lugha), hifadhi alama mbili za maoni kama zilivyo.
- Ikiwa unataka kuonyesha tu sehemu ya lugha, futa alama mbili za maoni na ondoa lugha yoyote ambayo hutaki kuorodhesha. Baada ya kuondoa alama hizo, Co-op Translator haitarekebisha sehemu hii moja kwa moja tena.

- Sehemu sasa inajumuisha ushauri "Ungependa Kuiga Kwenye Kompyuta Yako?" kusaidia watumiaji kuiga bila mzigo mkubwa wa tafsiri. Unaweza kubinafsisha ushauri huu kwa URL ya hifadhidata yako kwa kutekeleza, kwa mfano:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
