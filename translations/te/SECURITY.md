## Security

Microsoft మా సాఫ్ట్‌వేర్ ఉత్పత్తులు మరియు సేవల భద్రతను గర్భించి, ఇందులో మా GitHub సంస్థల ద్వారా నిర్వహించబడుతున్న అన్ని సోర్స్ కోడ్ నిల్వలు ఉంటాయి, ఇవి [Microsoft](https://github.com/Microsoft), [Azure](https://github.com/Azure), [DotNet](https://github.com/dotnet), [AspNet](https://github.com/aspnet) మరియు [Xamarin](https://github.com/xamarin) ను కలిగి ఉంటాయి.

మీరు ఏ Microsoft-స్వంతమైన నిల్వలో [Microsoft యొక్క భద్రతా లోపం నిర్వచనం](https://aka.ms/security.md/definition)కి తగిన భద్రతా లోపాన్ని కనుగొన్నారు అని భావిస్తే, దయచేసి క్రింద ఉన్న విధంగా మాకు తెలియజేయండి.

## Reporting Security Issues

**దయచేసి భద్రతా లోపాలను పబ్లిక్ GitHub Issues ద్వారా నివేదించవద్దు.**

దInsteadట మీ నివేదికలను Microsoft Security Response Center (MSRC) వద్ద [https://msrc.microsoft.com/create-report](https://aka.ms/security.md/msrc/create-report) వద్ద పంపండి.

లాగిన్ చేయకుండా సమర్పించాలనుకుంటే, [secure@microsoft.com](mailto:secure@microsoft.com) కి ఇమెయిల్ పంపండి. సాధ్యమైనట్లయితే, మా PGP కీతో మీ సందేశాన్ని ఎన్‌క్రిప్ట్ చేయండి; దయచేసి దానిని [Microsoft Security Response Center PGP Key page](https://aka.ms/security.md/msrc/pgp) నుండి డౌన్‌లోడ్ చేసుకోండి.

మీరు 24 గంటలలో స్పందన పొందాలి. ఏదైనా కారణంగా కాదైతే, దయచేసి మా ప్రాథమిక సందేశం పొందామని నిర్ధారించుకోడానికి ఇమెయిల్ ద్వారా అనుసరించండి. అదనపు సమాచారం [microsoft.com/msrc](https://www.microsoft.com/msrc) వద్ద లభిస్తుంది.

దయచేసి క్రింద పేర్కొన్న అవసరమైన సమాచారాన్ని (మీరు అందించగలంతవరకు) చేర్చండి, తద్వారా సమస్య స్వభావం మరియు పరిధిని మెరుగ్గా అర్థం చేసుకోవచ్చు:

  * సమస్య రకం (ఉదా. బఫర్ ఓవర్‌ఫ్లో, SQL ఇంజెక్షన్, క్రాస్-сైట్ స్క్రిప్టింగ్ మొదలైనవి)
  * సమస్య ప్రదర్శనకు సంబంధించిన సోర్స్ ఫైల్(ల) పూర్తి మార్గాలు
  * ప్రభావిత సోర్స్ కోడ్ యొక్క స్థానం (ట్యాగ్/బ్రాంచ్/కామిట్ లేదా డైరెక్ట్ URL)
  * సమస్యను పునరుత్పత్తి చేయడానికి అవసరమైన ఏదైనా ప్రత్యేక సెటప్
  * సమస్య పునరుత్పత్తి చేయడానికి దశల వారీ సూచనలు
  * ప్రూఫ్-ఆఫ్-కాన్సెప్ట్ లేదా ఎక్స్‌ప్లాయిట్ కోడ్ (సాధ్యమైతే)
  * సమస్య ప్రభావం, దానిని ఎలా ఒక దాడి యజమాని దోపిడి చేయవచ్చు

ఈ సమాచారం మా నివేదికను త్వరగా తగ్గించడానికి సహాయపడుతుంది.

మీరు బగ్ బౌంటీ కోసం నివేదిక ఇస్తున్నట్లయితే, పూర్తి నివేదికలు అధిక బౌంటీ అవార్డుకి సహాయపడతాయి. మా [Microsoft Bug Bounty Program](https://aka.ms/security.md/msrc/bounty) పేజీకి వెళ్లి మా ప్రస్తుత కార్యక్రమాల గురించి మరింత వివరాలు తెలుసుకోండి.
