# 🌐 బహుభాషా మద్దతు (Teamplate)

పరిచర్యాధారులు: కింద ఉన్న బ్లాక్ Co-op Translator ద్వారా నిర్వహించే "అన్ని భాషలు" ఉదాహరణ.

- మీరు `translate` (ఎలాంటి భాషా ఎంపిక) ని నడిపించినప్పుడు Co-op Translator ఈ జాబితాని పూర్తిగా నవీకరించాలని అనుకుంటే, రెండు వ్యాఖ్యా మార్కర్లు అలాగే ఉంచండి.
- మీరు కొంత భాషల జాబితా మాత్రమే చూపించాలనుకుంటే, రెండు వ్యాఖ్యా మార్కర్లను తొలగించి మీరు చూపించకూడని భాషలను తీసివేయండి.merkingలు తొలగించిన తర్వాత, Co-op Translator ఈ విభాగాన్ని ఆటోమేటిక్‌గా మార్చదు.

- ఇప్పుడు ఈ విభాగంలో "స్థానికంగా క్లోన్ చేసుకోవడం ఇష్టపడుతున్నారా?" అనే సలహా ఉంది, ఇది వినియోగదారులు భారీ అనువాద డేటాను లేకుండా క్లోన్ చేసుకోవడంలో సహాయపడుతుంది. మీరు ఈ సలహాను మీ రిపోజిటరీ URL తో వ్యక్తిగతీకరించవచ్చు, ఉదాహరణకు:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
