# 🌐 బహుభాషా మద్దతు (టెంప్లేట్)

నిర్వాహకులు: క్రింద బ్లాక్ Co-op Translator ద్వారా నిర్వహిస్తున్న "అన్ని భాషలు" ఉదాహరణ.

- మీరు `translate` (ఏ భాష ఎంపిక అయినా) నిర్వహించినప్పుడు Co-op Translator ఈ జాబితాను పూర్తి స్థాయిలో స్వయంచాలకంగా అప్‌డేట్ చేయాలని ఉంటే, రెండు కామెంట్ మార్కర్లను అలాగే ఉంచండి.
- మీరు కేవలం కొన్ని భాషలను మాత్రమే చూపించాలని ఉంటే, రెండు కామెంట్ మార్కర్లను తొలగించి, చూపించాలని కోరని భాషలను తీసివేయండి. మార్కర్లను తీసివేసిన తర్వాత, Co-op Translator ఈ విభాగాన్ని స్వయంచాలకంగా మార్చదు.

- ఇప్పుడు ఈ విభాగంలో "ప్రత్యక్షంగా క్లోన్ చేయడాన్ని మించినదా?" అనే సలహా ఉంది, ఇది పెద్ద అనువాద డేటాను లేకుండా క్లోన్ చేయడాన్ని సులభతరం చేస్తుంది. మీరు మీ రిపోజిటరీ URLతో ఈ సలహాను వ్యక్తిగతీకరించవచ్చు, ఉదాహరణకు:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
