# మద్దతు పొందిన భాషలు

కింద ఉన్న పట్టికలో ప్రస్తుతం **Co-op Translator** మద్దతు ఇస్తున్న భాషలు సూచించబడ్డాయి. ఇందులో భాషా కోడ్లు, భాషా పేర్లు మరియు ప్రతి భాషకు సంబంధించిన ఏమైనా తెలిసిన సమస్యలు ఉంటే అవి ఉంటాయి. మీరు కొత్త భాషకు మద్దతు ఇచ్చాలనుకుంటే, దయచేసి సంబంధిత భాషా కోడ్, పేరు మరియు సరైన ఫాంట్ను `src/co_op_translator/fonts/`లో ఉన్న `font_language_mappings.yml` ఫైలులో జతచేసి పరీక్షించిన తరువాత pull request సమర్పించండి.
