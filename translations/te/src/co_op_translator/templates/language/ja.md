జపనీస్ మోడ్: Markdown టోకెన్లను ఖచ్చితంగా సంరక్షించాలి.

నియమాలు (అనుసరించాలి):
1) Markdown లింకులను పూర్తిగా అలాగే ఉంచాలి: [text](../../../../../../src/co_op_translator/templates/language/URL) -> [translated text](../../../../../../src/co_op_translator/templates/language/అదే URL).
2) లింకులను నేరుగా వచనం లాగా మార్చవద్దు (ఉదా: 「text」（URL）, text (URL)).
3) కేవలం లింక్ టెక్స్ట్‌ను మాత్రమే అనువదించాలి; Markdown నిర్మాణం మరియు URL మార్పు కాకూడదు.
4) Markdown లింక్ చుట్టూ 「」 పెట్టరాదు, అంతర్గత వ్యాకరణం అవసరం కారణంగా కాకుంటే.

సంరచన శైలికంటే ముఖ్యం.
