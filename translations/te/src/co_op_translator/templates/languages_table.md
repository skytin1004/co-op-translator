>
> ఈ రిపోజిటరీ 50+ భాషల భాషಾಂತರాలనున్నది, ఇది డౌన్లోడ్ పరిమాణం గణనీయంగా పెంచుతుంది. భాషాంతరాలు లేనివిగా క్లోన్ చేయడానికి, sparse checkout ఉపయోగించండి:
>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse <repo_url>.git
> cd <repo_name>
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse <repo_url>.git
> cd <repo_name>
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> ఇది కోర్సును పూర్తి చేసుకోవడానికి అవసరమైన ప్రతిఒక్క దానిని మీరు much వేగంగా డౌన్లోడ్ చేస్తారు.
