# समर्थित भाषाएँ

नीचे दी गई तालिका में वर्तमान में **Co-op Translator** द्वारा समर्थित भाषाओं की सूची है। इसमें भाषा कोड, भाषा के नाम, और हर भाषा से जुड़ी ज्ञात समस्याएँ शामिल हैं। यदि आप किसी नई भाषा के लिए समर्थन जोड़ना चाहते हैं, तो कृपया संबंधित भाषा कोड, नाम, और उपयुक्त फॉन्ट को `src/co_op_translator/fonts/` में स्थित `font_language_mappings.yml` फ़ाइल में जोड़ें और परीक्षण के बाद एक pull request सबमिट करें।
