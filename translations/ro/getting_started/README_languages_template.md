# 🌐 Suport Multilingv (Șablon)

Menținători: Blocul de mai jos este un exemplu „toate limbile” gestionat de Co-op Translator.

- Dacă doriți ca Co-op Translator să mențină această listă complet actualizată automat când rulați `translate` (orice selecție de limbă), păstrați cele două marcatoare de comentarii așa cum sunt.
- Dacă doriți doar să afișați un sub-set de limbi, eliminați cele două marcatoare de comentarii și ștergeți limbile pe care nu doriți să le listați. După eliminarea marcatoarelor, Co-op Translator nu va mai înlocui automat această secțiune.

- Secțiunea include acum un sfat „Preferi să Clonezi Local?” pentru a ajuta utilizatorii să cloneze fără încărcătura mare a traducerilor. Puteți personaliza acest sfat cu URL-ul repository-ului dvs. rulând, de exemplu:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
