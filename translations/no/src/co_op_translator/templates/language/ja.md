# Oversettelse med Co-op Translator

Denne veiledningen vil hjelpe deg å bruke [Co-op Translator](https://github.com/Azure/co-op-translator) til effektivt oversette og korrekturlese innholdet i PRer.

## Komme i gang

1. Installer [Co-op Translator](https://github.com/Azure/co-op-translator) som en GitHub App.
2. Når appen er aktiv, gå til PRseksjonen for å finne oversettelsesmuligheter.

## Oversette

- Klikk på "Oversett innhold" i PR-detaljene.
- Velg målspråk.
- Editor-panelet åpnes med originalteksten og et oversettelsesforslag.

## Korrekturlese

- Les gjennom den foreslåtte oversettelsen.
- Rediger teksten om nødvendig.
- Lagre endringene for å oppdatere PRen.

## Tips

- Sjekk spesielt tekniske termer for korrekthet.
- Bruk formatet i originalen for å bevare struktur og tydelighet.

Få mer informasjon i prosjektets [GitHub repository](https://github.com/Azure/co-op-translator).
