# Støttede språk

Tabellen nedenfor viser språkene som for øyeblikket støttes av **Co-op Translator**. Den inkluderer språk-koder, språknavn, og eventuelle kjente problemer knyttet til hvert språk. Hvis du ønsker å legge til støtte for et nytt språk, vennligst legg til tilsvarende språk-kode, navn og passende font i `font_language_mappings.yml` filen som ligger i `src/co_op_translator/fonts/` og send en pull request etter testing.
