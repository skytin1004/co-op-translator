# Podržani jezici

Tablica u nastavku prikazuje jezike koje trenutno podržava **Co-op Translator**. Uključuje jezične kodove, nazive jezika i poznate probleme povezane s pojedinim jezicima. Ako želite dodati podršku za novi jezik, dodajte odgovarajući jezični kod, naziv i odgovarajući font u datoteku `font_language_mappings.yml` koja se nalazi u `src/co_op_translator/fonts/` i pošaljite pull request nakon testiranja.
