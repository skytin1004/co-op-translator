# Uvod u Localizeflow

Localizeflow je moćan alat koji pomaže programerima i timovima da lako upravljaju prijevodima svojih projekata. Pomaže vam da centralizirate prijevode, automatizirate lokalizacijske procese i osigurate da vaš softver bude dostupniji globalnoj publici.

[!TIP]
Koristite Localizeflow za učinkovito upravljanje svojim prijevodima kroz integraciju s GitHub App.

## Ključne značajke Localizeflow

- Centralizirano upravljanje prijevodima
- Integracija s GitHub App za jednostavniju suradnju
- Automatska sinkronizacija prijevoda putem PR-ova
- Podrška za višestruke jezike i formate datoteka

# Početak rada

Za početak, instalirajte [GitHub App](https://github.com/apps/localizeflow) Localizeflow u vaš repozitorij. Nakon što je instaliran, moći ćete kreirati i upravljati prijevodima izravno unutar svog repozitorija.

## Kreiranje prvog prijevoda

1. Otvorite Localizeflow sučelje u GitHub App.
2. Dodajte jezik na koji želite prevesti vaše datoteke.
3. Kreirajte PR (pull request) s prijevodima.
4. Nakon pregleda i odobrenja PR-a, novi prijevodi bit će automatski spojeni u baznu granu.

[!NOTE]
Pratite napredak prijevoda i pregledajte moguće probleme direktno u PR-ovima kako bi održali visoku kvalitetu prijevoda.

# Daljnji resursi

Za dodatnu pomoć i primjere, pogledajte službenu dokumentaciju Localizeflow na [Localizeflow Docs](https://localizeflow.io/docs).

Ako imate pitanja ili prijedloge, pridružite se našoj zajednici ili otvorite pitanje na [Localizeflow GitHub Issues](https://github.com/localizeflow/localizeflow/issues).

Uživajte u učinkovitom upravljanju prijevodima s Localizeflow!
