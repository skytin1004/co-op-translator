# 🌐 Wsparcie Wielojęzyczne (Szablon)

Opiekunowie: Poniższy blok to przykład „wszystkich języków” zarządzany przez Co-op Translator.

- Jeśli chcesz, aby Co-op Translator automatycznie utrzymywał tę listę w pełni aktualną za każdym razem, gdy uruchamiasz `translate` (wybór dowolnego języka), zachowaj dwa znaczniki komentarza bez zmian.
- Jeśli chcesz pokazać tylko podzbiór języków, usuń dwa znaczniki komentarza i usuń języki, których nie chcesz umieszczać na liście. Po usunięciu znaczników, Co-op Translator nie będzie już automatycznie zastępował tej sekcji.

- Sekcja zawiera teraz poradę „Wolisz sklonować lokalnie?”, która pomaga użytkownikom sklonować repozytorium bez dużej paczki tłumaczeń. Możesz spersonalizować tę poradę, podając adres repozytorium, uruchamiając na przykład:
  - `translate -l "ko" --repo-url "https://github.com/org/repo.git"`
